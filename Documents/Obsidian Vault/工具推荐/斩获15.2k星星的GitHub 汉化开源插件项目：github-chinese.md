## 完整内容

对于刚接触 GitHub 的用户来说，原本全英文的菜单、按钮和标题可能造成使用上的障碍。为了解决这个问题，GitHub 中文化插件 提供了一种轻量、高效的解决方案。这个开源用户脚本能够将 GitHub 网站的公共组件自动翻译成简体或繁体中文，适合想要更流畅上手 GitHub 的“小白”用户使用，尤其在本地安装后即可生效，无需繁琐配置。

GitHub 中文化插件 是什么？

GitHub 中文化插件 是一个由 maboloshi 发起的开源用户脚本项目，旨在实现 GitHub 界面的全面中文化，包括菜单栏、标题、按钮等常见组件。项目基于 GPL-3.0 协议发布，支持简体中文与繁体中文翻译，并采用智能正则匹配与持续更新的词库机制，具体定位和功能请以仓库为准。

核心功能

为不同语言能力的 GitHub 用户提供视觉友好的中文界面，降低使用门槛，核心功能包括：

界面元素全面中文化——自动翻译菜单栏、标题、按钮等 GitHub 公共组件。

智能正则匹配——更准确地定位并翻译界面文本。

项目描述翻译支持——提供人机结合的项目简介翻译。

时间元素本地化——自动转换显示时间为本地中文格式。

持续词库更新——包含活跃的词条更新机制，保证翻译质量与准确度。

操作指南

以下步骤帮助用户快速安装并使用该插件：

打开开源页面下载插件：https://github.com/maboloshi/github-chinese

安装用户脚本管理器，如 Tampermonkey 或 Violentmonkey。

进入插件仓库或 GreasyFork 页面，点击「安装脚本」。

安装完成后，访问 GitHub 页面即可看到界面自动显示为中文。

可在脚本管理器中选择“开发版”（实时更新）或“稳定版”。

开发版：每周五自动更新词库。

稳定版：每周一同步词库，更新节奏更稳。

支持平台

该插件是通用用户脚本，支持多种浏览器与脚本管理器：

Chrome / Chromium 内核浏览器：兼容 Tampermonkey、Violentmonkey。

Safari（全平台）：支持 Tampermonkey、Macaque、Stay。

Firefox / Gecko 内核浏览器：支持 Tampermonkey、Violentmonkey。

Via（Android 浏览器）：内置脚本管理器也支持使用。

常见问题

Q1：安装后是否需要注册或登录才能生效？

A：前提是已安装用户脚本管理器，访问 GitHub 页面即可自动生效，无需注册。

Q2：是否影响 GitHub 的原功能或安全性？

A：插件只在本地修改界面文本，不涉及后台数据操作，安全性较高。

Q3：可以切换中英文界面吗？

A：插件默认根据设置自动翻译，但可以在脚本管理器中启用/禁用，切换语言界面。
