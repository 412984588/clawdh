## 完整内容

在日常获取网盘资源时，开发者和资源爱好者常面临搜索来源分散、速度慢、结果杂乱等问题。比如想找某部电影的网盘链接，需在多个平台逐个搜索，还得手动筛选有效链接，耗时又费力。而Pansou作为高性能、可扩展的网盘资源搜索API服务，能通过Telegram频道和自定义插件实现多源并发搜索，还具备智能排序、网盘分类等功能，轻松解决这些痛点。你可以通过Pansou[1]获取开源代码，快速搭建属于自己的资源搜索系统。

Pansou是什么？

Pansou定位为高性能、可扩展的网盘资源搜索API服务，专为开发者和资源爱好者设计。它基于Go语言开发，支持通过Telegram频道和自定义插件进行多源并发搜索，拥有智能排序、网盘类型分类、异步插件系统及二级缓存机制，还可通过Docker一键部署前后端，方便开发者快速构建资源搜索系统，具体功能细节可参考GitHub仓库及官方文档。

核心功能

Pansou的核心价值在于为用户提供高效、精准的网盘资源搜索能力，重点服务于需要快速获取网盘资源的开发者、资源爱好者，以及需搭建资源搜索系统的团队，大幅提升资源搜索与整合效率。

•多源并发搜索——支持Telegram频道与自定义插件双来源搜索，并发执行多个搜索任务，显著提升搜索速度，缩短资源获取时间。
•网盘类型自动分类——能自动识别百度网盘、阿里云盘、夸克网盘等10余种网盘链接，按类型归类展示，方便用户快速筛选目标资源。
•多维度智能排序——基于插件等级、时间新鲜度和优先关键词的综合排序算法，让优质、最新的资源排在前列，提升搜索结果有效性。
•异步插件系统——支持通过插件扩展搜索来源，采用“尽快响应，持续处理”模式，解决部分搜索源响应时间长的问题，同时便于灵活扩展功能。
•二级缓存机制——采用分片内存+分片磁盘缓存，大幅提升重复查询速度与并发性能，减少重复请求，降低服务器压力。
•MCP服务集成——提供基于Model Context Protocol的服务，可将搜索功能集成到Claude Desktop等支持MCP的应用中，拓展使用场景。
•Docker一键部署——支持Docker及Docker Compose部署，前后端集成版开箱即用，无需复杂配置，开发者可快速启动服务。
•灵活参数配置——支持通过环境变量配置服务端口、并发数、缓存有效期等参数，满足不同用户的个性化使用需求。

操作指南

Pansou新手可通过简单步骤快速上手，无论是直接使用服务还是部署搭建，都能在短时间内完成基础操作。

1.若需直接体验服务，可先访问Pansou开源地址（https://github.com/fish2018/pansou），了解基础功能与API文档（注意：建议先查看健康检查接口，确认服务是否正常）。[2]
2.若选择Docker一键部署前后端集成版，打开终端，输入指令“docker run -d --name pansou -p 80:80 ghcr.io/fish2018/pansou-web”并执行，等待服务启动。
3.若需部署纯后端API，使用Docker命令“docker run -d --name pansou -p 8888:8888 -v pansou-cache:/app/cache -e CHANNELS="tgsearchers3,xxx" ghcr.io/fish2018/pansou:latest”，或通过Docker Compose部署（推荐）。
4.服务启动后，可通过“http://localhost:80”（前后端集成版）或“http://localhost:8888”（纯后端API）访问，调用「/api/health」接口检查服务状态。[3]
5.调用搜索API时，可选择POST或GET方法，按文档要求传入“kw”（搜索关键词）等必要参数，如POST请求可传入包含关键词、目标网盘类型的JSON数据。
6.查看搜索结果，根据“merged_by_type”字段按网盘类型筛选资源，获取有效链接与密码（注意：需验证链接有效性，避免无效资源）。
7.若需自定义配置，可通过环境变量修改服务端口、并发数、缓存有效期等参数，如设置“PORT=8889”修改服务端口。
8.若需扩展搜索来源，可参考插件开发指南开发自定义插件，或在配置中指定启用特定插件。

常见问题

•Q：使用Pansou搜索网盘资源是否安全，是否会泄露个人信息？ A：Pansou作为开源API服务，本身不存储用户搜索记录或个人信息，搜索过程主要基于公开的Telegram频道与插件来源。但需注意，获取的网盘链接可能存在安全风险，建议验证资源安全性后再使用，同时避免在请求中传入敏感信息。
•Q：Pansou支持哪些网盘类型，是否可以新增未支持的网盘类型？ A：目前Pansou支持百度网盘、阿里云盘、夸克网盘、天翼云盘等10余种网盘类型及磁力链接、电驴链接。若需新增网盘类型，可通过开发自定义插件实现，具体可参考项目插件开发相关文档，或提交功能需求至GitHub Issues。
•Q：从源码安装Pansou需要哪些环境，安装过程中遇到问题如何解决？ A：从源码安装需具备Go 1.18+环境，可选SOCKS5代理（访问Telegram站点时）。安装中若遇依赖缺失、编译错误等问题，可参考GitHub仓库的“快速开始”文档，或在Issues中搜索类似问题，也可提交新Issue获取社区帮助。
