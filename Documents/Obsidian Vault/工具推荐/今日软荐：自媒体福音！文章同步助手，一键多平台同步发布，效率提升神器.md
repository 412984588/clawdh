## 完整内容

做自媒体的都很能够共情的一点：

我们常常需要在多个平台上发布内容：头条、知乎、CSDN、简书、公众号。。。

如果每个平台都要重新排版、复制粘贴，实在让人头疼！

推荐一款我一直在用的极简高效工具：文章同步助手，它可以帮你一键同步到23+个平台，让编辑发布变得轻松快捷！

并且它还是一个开源项目，本人用了两年多还一直在更新优化中。。。虽然有些文字同步到某平台的排版有些鸡肋，但还是无法舍弃它。

功能亮点

• 多平台同步 ：支持头条、WordPress、知乎、简书、掘金、CSDN 等 23+ 平台！在网页上选择要同步的文章内容，通过右键菜单的提取正文功能，即可在各大平台间同步，实现一键发布，真正做到“编辑一次，发布到位”。

• 微信公众号同步 :想要把其他平台的优质内容搬到公众号？简单！在浏览器中打开任意公众号文章，点击左上角的“同步该文章”按钮，选择发布渠道，即可实现公众号的内容同步。

• Markdown 一键发布 :支持从在线 Markdown 编辑器同步到知乎、头条等发布渠道。Markdown 格式排版清晰，能够保证内容在多平台间无损展示。

安装与使用步骤

1、下载和安装

从官网、谷歌应用商店或 GitHub 获取最新版本的文章同步助手，按说明添加到浏览器扩展。

如有下载及访问不了的问题，也可回复 文章同步助手 获取离线包。

解压后安装浏览器扩展文件安装方式：将.crx文件拖动到浏览器中 或 在浏览器扩展页面-添加扩展-选择解压后的文件夹自动安装。

2、内容同步操作

• 右键同步：在网页上右键点击插件提供的“提取正文”功能，选择同步到的平台，即可实现一键发布。

• 公众号同步：打开目标公众号文章，点击同步按钮，选择同步渠道，等待同步完成即可。

• Markdown 发布：在 Markdown 编辑器中编辑内容，选择同步发布渠道，系统会自动完成格式转换和发布。
