Czkawka是什么

Czkawka（读作 “恰卡夫卡”，意为“打嗝”）是一款功能强大的开源工具，用于清理电脑中的无用文件。它可以帮助你快速找出重复文件、空文件夹、相似图片、损坏的文件等，彻底清理你的磁盘空间！最棒的是，它使用Rust语言开发，运行速度飞快，支持多平台（Windows、Linux、macOS等），而且完全无广告、无追踪，给你一片清爽的数字世界。

开源成就
Star数  GitHub 上这款神器已经斩获了 24.2k Star。
主开发语言Rust，这门以“性能与内存安全”为卖点的语言，使得 Czkawka 高效、稳定又安全。
核心功能
寻找重复文件： 通过文件名、大小和内容哈希值，迅速定位磁盘中多余的重复文件，不再浪费宝贵空间。
清除空文件夹： 一键扫描，删除没用的空文件夹，保持文件系统整洁。
找出相似图片： 通过图像特征对比，识别分辨率不同但内容相似的图片，即使加了水印也能发现。
视频清理： 支持查找视觉上相似的视频（是不是很神奇！）。
识别坏文件： 定位损坏或无法正确读取的文件，让你的数据管理更加可靠。
无效符号链接： 找出指向不存在的文件/文件夹的符号链接轻松修复。
清理垃圾文件： 删除临时文件、不良扩展名文件等，让磁盘告别“杂乱无章”。
支持多语言和离线： 内置各种语言界面，还能离线使用，保护你的隐私安全。
安装指南

Czkawka 支持命令行和图形界面两种模式，方便不同习惯的用户操作。以下是简要使用步骤：

安装：

Windows/macOS/Linux 用户可直接前往 GitHub Releases页面，下载对应系统的安装包。

或者通过命令安装（以 Linux 为例，需安装 Cargo）：

cargo install czkawka_gui


使用：

图形界面：运行应用程序后，选择需要清理的项目（如“空文件夹”或“重复文件”）即可。
命令行：对于开发者，运行以下命令可快速清理（以找重复文件为例）：
czkawka_cli duplicates /path/to/scan


Czkawka 用简单快速的方式帮你清理电脑文件，“无用文件终结者”这个称号绝对实至名归！得益于 Rust 的优异性能，它既轻量又高效，无广告、无隐私问题，还能跨平台使用。如果你曾因硬盘空间不足烦恼，这款工具无疑是你不可错过的选择。
