ConvertX 是什么

ConvertX 是一个完全开源、基于 TypeScript 和 Bun 开发的自托管文件转换工具。它允许用户在本地运行一个类似在线工具的平台，可以高效、灵活地将文件从一种格式转换为另一种格式。你不需要再去找各种转换网站，也不用因为担心文件隐私而倍感焦虑！只要搭个环境，它随时“开机即用”，吊到飞起。

强大的扩展性： 基于多种顶级工具和库，如 ImageMagick、FFmpeg、Calibre 等，可拓展性超强！

安装指南

简单几步，几分钟内就可以搭建起这个神器：

1.安装 Docker（如果还没有安装）：
sudo apt-get update
sudo apt-get install docker docker-compose

2.下载并运行 ConvertX：
在项目目录中运行以下命令：
docker run -p 3000:3000 -v ./data:/app/data ghcr.io/c4illin/convertx

3.访问本地服务：
在浏览器中输入以下地址，开始使用：
http://localhost:3000  

4创建账户并配置：
创建管理员账号，并根据需要设置访问权限，就可以开始愉快地转换文件了！
