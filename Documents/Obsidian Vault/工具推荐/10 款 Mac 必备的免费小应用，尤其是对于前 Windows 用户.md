## 完整内容

几年前，我决定在工作上不再跟随传统，放弃换另一台 Windows 笔记本的念头，而是要勇敢地走进苹果的世界，成为“酷炫一族”中的一员。对于我这样一位20多年长期沉浸在 Windows 操作系统中的用户来说，这无疑是一次全新的挑战。

虽然 Windows 中的很多问题都能迎刃而解，但切换到 MacBook（更准确地说是 macOS）后，面对的那些细微而不同的难题却层出不穷。对于那些从小就习惯 macOS 的用户来说，很多“怪异”之处可能早已成为生活的一部分；但对我而言，原有的 Windows 惯性根深蒂固，想要彻底改掉就像改写内心的密码一样艰难。为了让这段过渡期不那么艰苦，我特地找来了一系列小工具，帮助我解决在使用 macOS 时碰到的那些“难题”。更让人心动的是，这些工具全都是免费的！

AltTab
❝

https://alt-tab-macos.netlify.app/

macOS 的窗口管理一直是我心中的一块短板。尽管系统最近才加入了窗口捕捉功能，但在窗口间切换的体验仍然远不如 Windows。Windows 上那简洁明了的 Alt+Tab 多年来一直让我得心应手；而 macOS 默认的 Cmd+Tab 则只会在应用之间切换，远远没有达到我预期的窗口级别切换体验。

在 Windows 中，你只需按下 Alt+Tab，便能迅速在所有窗口间切换；而在 Mac 上，如果想要在同一应用内切换多个窗口，你往往得借助右键点击 Dock 图标、使用 App Exposé 或者调用完整的 Mission Control——这些操作不仅麻烦，而且对于习惯了键盘快捷键操作的我来说，实在不够“友好”。

AltTab 正是为了解决这一困扰而诞生。它允许你自定义快捷键（甚至可以同时设置多达五种不同的切换方案），并能调整各种细节设置。我的配置非常简单：我习惯于一次切换所有窗口，无论是分布在不同桌面、全屏状态下的窗口，还是隐藏或最小化的窗口都能一并管理。你只需看着那清晰的设置界面，心中便能感受到那种 Windows 式的流畅与高效。
例如，我将“快捷键1”的暗灰区域设为实际使用的切换键，并取消了“鼠标悬停”这一选项。

Battery Toolkit
❝

https://github.com/mhaeuser/Battery-Toolkit

在 Windows 或 macOS 中，我们都无法找到原生支持的电池充电上限设置功能。对于我这样常常插着电源工作的用户来说，缺乏定时充电的功能让我倍感头疼。

Battery Toolkit 就是这样一款简单实用的工具，它允许你手动设定电池充电的最大限度。想象一下，当你明知道接下来一段时间无法接通电源时，只需轻轻一点，就能让电池充满电备用，这样的贴心设计无疑解决了我的燃眉之急。
只需简单配置几下，你就能轻松掌控充电策略。

Clocker
❝

https://apps.apple.com/us/app/clocker/id1056643111?mt=12

如果你的工作需要跨越多个时区，那么 macOS 自带的小部件可能会让你失望：那种点击时右侧弹出的时钟小窗口远不如 Windows 那种直观明了。

Clocker 则完全颠覆了这种体验。它在菜单栏中设置了一个图标，点击之后便能显示出你自己挑选的各个时区的时间。
除此之外，它还拥有一个叫做“时间滑动条”的功能，这个功能让我大为惊喜：只需拖动时间轴，便能轻松计算出未来或过去某个时刻的时间。虽然我本人并不常用设置提醒或显示日历事件，但对于我来说，Clocker 就是那种简洁而实用的跨时区助手。

Command X
❝

https://apps.apple.com/us/app/command-x/id6448461551?mt=12

作为一个长年沉浸在 Windows 世界中的用户，Ctrl+C、Ctrl+V 和 Ctrl+X 已经深深烙印在我的肌肉记忆里。转而使用 Mac 后，这种习惯不得不面对从 Control 键到 Command 键的转变。

问题在于，在 Windows 中，用 Ctrl+X 剪切文件在资源管理器中司空见惯，而 macOS 则有着截然不同的处理方式。Mac 并没有“剪切”这一操作，而是采用了“剪切后粘贴并删除原文件”的方式，这一切都需要使用 Cmd+Option+V 来完成。
为了省却记忆这一全新操作方式的麻烦，我选择了 Command X 这款工具，它能够在 Finder 中添加出与 Windows 相似的快捷键（Cmd+X），让你无需改变原有习惯，就能轻松完成文件的剪切操作。

Ice
❝

https://icemenubar.app/

菜单栏在 macOS 中就像 Windows 的系统托盘一样，承载着各种系统功能图标。但与 Windows 不同的是，macOS 内建的设置项往往不能满足我们对菜单栏图标个性化管理的需求。


市面上虽然有诸如 Bartender 这样的专业工具来整顿菜单栏，但它毕竟是收费软件。Ice 则成为我理想的免费替代品。
它为我提供了三种模式来管理菜单栏图标：

始终显示——针对那些必不可少的图标；
点击显示——适用于偶尔需要的功能；
右键显示——对那些几乎不需要的图标，通过 Option+点击来唤出。
更妙的是，你可以选择将这些图标以两种形式展现：一种是直接在菜单栏内扩展显示，另一种则是在菜单栏下方单独浮动一个二级菜单栏。
在拥有额外大屏幕（如外接显示器或 iMac）的环境下，直接在菜单栏内扩展显示非常方便；但对于带有刘海的 MacBook，系统会自动隐藏刘海附近的图标，这时一个单独的菜单栏就显得尤为重要。
通过 Ice，我可以将常用的图标始终保留在主菜单栏上，而其他的则折叠在一个二级菜单中，使整个桌面看起来既整洁又高效。
LinearMouse
❝

https://linearmouse.app/zh-CN/

这是我最近发现的一款神奇工具，它弥补了 macOS 在鼠标缩放体验上的不足。在 Windows 中，按住 Control 键并滚动鼠标滚轮便可轻松实现网页或图片的缩放；然而在 Mac 上，这样的操作根本不存在。

通常，你只能依赖应用程序菜单中提供的缩放选项，或使用触控板上的捏合手势来放大缩小页面。LinearMouse 则解决了这一困扰：通过设置，它能让 Control 键充当缩放修饰键，使得滚动鼠标滚轮也能实现缩放效果。


虽然它在浏览器中不能直接改变网页的缩放级别，而只是调用了 macOS 自带的缩放功能（因此无法通过菜单恢复原始比例），但对于绝大多数情况下的使用来说，已经足够应付日常需求。

Maccy
❝

https://maccy.app/

在 Windows 中，按下 Win+V 即可调出剪贴板历史记录，而 macOS 却没有内建类似功能。作为一个习惯了 Windows 剪贴板管理的用户，我迫切需要一款能够记录图片和文件的剪贴板管理器。

Maccy 正是我经过多方比较后选定的工具。它驻留在菜单栏中，一点击图标便能显示出你最近的剪贴板记录，你也可以设置快捷键来实现类似 Windows 的操作体验。
此外，它还允许你为特定应用设定忽略列表（例如那些密码管理器），这样就不会误把敏感信息记录下来。再加上一个贴心的“去除格式化”功能，当你粘贴文本时能自动剔除多余格式，让粘贴操作变得更加干净简洁。

Middle Click

（注意市面上有多个同名应用，这里请通过图标区分）

❝

https://github.com/artginzburg/MiddleClick

在 Windows 中，如果你想用鼠标中键点击链接自动在新标签页中打开，只需在触控板上用三根手指轻点即可轻松实现。但在 macOS 中，这种直观的体验却不存在。

为了解决这一问题，我安装了 Middle Click。这个工具可以模拟鼠标中键点击，确保你在浏览器或其他支持中键功能的应用中，能以最自然的方式打开新标签页。


它的设置极其简单，你甚至可以选择是否只在触控板轻点时生效，或是在真正按下去时响应，就这么简单直接。

Mini Calendar
❝

https://apps.apple.com/us/app/mini-calendar/id1088779979?mt=12

在日常工作和社交中，有时我们只需要迅速看一眼日期，而不用打开复杂的日历程序。在 Windows 中，你只需点击时钟图标，一个小巧的日历便会弹出，轻松满足需求。



而 macOS 提供的那种静态日历部件只能显示当前月份，并且要切换月份就必须打开完整的日历应用，这无疑打断了思路。


Mini Calendar 就是为此而生——一个仿 Windows 风格的迷你日历，它可以直接从菜单栏中弹出，快速浏览各个月份，让你在紧张的工作中也能随时掌握日期信息。
只需轻轻一按，日历便会如画卷般展开，而点击其中的点又能迅速返回当前日期，既简单又高效。

Shottr
❝

https://shottr.cc/

无论在 Windows 还是 macOS 中，系统自带的截屏工具都还算不错，但我总觉得还可以更进一步。于是，我找到了 Shottr——一款功能丰富的截屏工具。


虽然 macOS 自带的截屏工具不仅能拍照还能录屏，但我却更钟情于 Shottr 的多样化操作。它不仅可以进行传统的调整（如绘图、裁剪），还能添加热区计数器，甚至能够模糊或直接删除图像中的文字。



需要说明的是，Shottr 实际上不是完全免费的，而是一款永久分享软件。它在使用时会偶尔弹出购买完整版的提示，但我始终觉得这款工具的实用性远超那点小烦恼——于是，尽管心有一丝犹豫，我还是选择了保留这份“免费”的使用方式。
