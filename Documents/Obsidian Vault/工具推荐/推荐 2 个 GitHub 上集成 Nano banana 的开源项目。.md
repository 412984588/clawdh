01

香蕉超市

这个叫做 Nano Bananary （香蕉超市）的开源项目，是一个零提示词图像生成工具，开源三周就获得 1000 多的 Star 了。


不用费心思想提示词，就能轻松处理图片，对新手特别友好。

它的用法很简单，选好想要的玩法、上传图片后，点一下生成就能出效果，还能局部涂选修改细节。

更方便的是，每次生成的图片都能直接当下次编辑或生成的素材，整个流程很顺畅。

目前它有不少实用效果，比如能把图片场景改成漂亮的黄金时刻日落氛围，也能把图片贴到汽水罐上做成精致的产品图，还有和冒险角色相关的设计功能，满足不同的图片处理需求。

而且，现在推出了视频生成功能。

如何部署使用

① 准备环境

安装 Node.js；申请 Google Gemini API Key（在 https://aistudio.google.com/ 获取）

② 部署项目

# 克隆代码
git clone https://github.com/ZHO-ZHO-ZHO/Nano-Bananary.git
cd Nano-Bananary
# 安装依赖
npm install
# 配置API密钥 (在项目根目录创建 .env.local 文件，并填入)
echo "VITE_GEMINI_API_KEY=你的密钥" > .env.local

③ 运行使用

# 启动项目
npm run dev # 用浏览器打开终端显示的本地地址即可使用

Nano Banana 捏脸神器

BananaFace 是一款好用的开源捏脸神器，也叫香蕉造型师，主打高人物一致性，能轻松帮人捏脸、做美容效果。

还能打造大师级的人物造型和肖像，它的 slogan 是 你的随身造型师。

用它处理人物形象很灵活，有文生图和图生图两种模式可选。

而且不用自己琢磨填提示词，靠选参数就能调整效果，总共包含 4 大类 44 项专业参数，比如能选肖像类型、调整年龄和种族，也能定脸型、表情。

像选 “开心” 的表情，挑 “蓝色” 的瞳色，选 “不对称剪裁” 的发型、“灰色” 的发色都可以

就连摄影相关的参数也能调，比如相机类型、镜头类型、焦距，还有灯光方向（比如从左上方打光）、柔和环境光等，能细致把控最终的肖像效果。
