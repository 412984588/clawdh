06 接入无问芯穹 API

首先注册登录：https://cloud.infini-ai.com/

这个平台非常直接，登录进去以后就让“创建API Key”

名称随便输入，关键是创建之后，点密钥前面的复制图标

根据要求输入验证码，就可以复制密钥了

打开CherryStudio，在模型服务里找到无问芯穹，把密钥粘贴进去。

检查的时候选“deepseek-r1”，确认成功连接即可。

07 接入SophNet API

注册登录：https://www.sophnet.com/#/login

登录之后，里面有一个默认创建的项目，点击“详情”

进入详情页面后，选择“添加新服务”，设置如下图

点击左边的“API Key”，创建新的API Key

创建之后，复制一下，到CherryStudio，如果模型列表里没找到SophNet，需要手动添加。

在模型服务列表最底部的“添加”，服务商名称输入“SophNet”，提供商类型选“OpenAI”

确认之后，再粘贴刚才复制过的密钥进去。

API那里完整输入下面【】内的链接（一点都不要错！）
【https://www.sophnet.com/api/open-apis/chat/completions#】

然后在模型那里点“添加”
输入模型ID“DeepSeek-R1”（必须完全一致！）

最后点“检查”确认能正常连接就可以了。
