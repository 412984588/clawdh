释放 AI 潜力

AIPyApp 做了一个很酷的事情：为 AI 集成了一个 Python 运行环境，会根据你的意图自动写 Python 代码并执行，释放 AI 大模型的全部潜力。

相当于给 AI 装上双手，你只要说出你的想法，就能帮你分析本地数据，操作本地应用，给你最终结果。

而且这个开源项目有一个配套的客户端：爱派（AiPy），所有细节都帮你封装好了，你只需要下达命令就好。

我安装了一下这个客户端，测了几个 Case 还是挺有意思的。

如何部署

① 使用 AIPython

AIPython 有两种运行模式：

任务模式：非常简单易用，只需输入你的任务，适合不熟悉 Python 的用户。
Python 模式：适合熟悉 Python 的用户，允许同时输入任务和 Python 命令，适合高级用户。

② 使用爱派

你可以直接访问这个开源项目对应的官网，去下载对应的客户端。现在注册，填写我的邀请码：G6qw，还能免费额外领取150万Tokens。

安装后可以用平常说话的方式告诉 AiPy 要做什么。

它能把你的话翻译成代码并执行，大大简化了使用 Python 处理各种任务的过程，特别是数据处理和自动化方面，让工作高效、自然。

而且从 Python 编程语言诞生以来，就有各种各样的工具包，这些工具都将成为 AI 的弹药库，AiPy 让 AI 可以调用各种 Python 工具包来满足你的诉求。
