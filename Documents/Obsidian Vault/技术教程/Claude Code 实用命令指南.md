## 内容摘要
这篇文章介绍了Claude Code的9个实用命令，这些命令能够显著提升开发效率，包括项目初始化、上下文管理、思考深度控制等功能。

## 核心命令详解

### 1. 项目管理命令
- **/init**: 让Claude读取整个项目，生成cloud.md文件，为后续对话提供完整的项目背景
- **/clear**: 清空对话历史，适合开始新任务前使用，避免上下文干扰
- **/reload**: 重新加载配置或记忆文件

### 2. 上下文优化命令
- **/compact**: 压缩对话上下文，丢弃不重要的历史内容，节省token使用量，让AI更专注

### 3. 思考控制命令
- **/think**: 基础思考模式
- **/thinkhard**: 深度思考模式
- **/thinkharder**: 更深度思考模式
- **/ultrathink**: 超深度思考模式
  - 使用建议：任务难度越高，使用更强的思考模式

### 4. 实用工具命令
- **/help**: 查看所有可用命令
- **/status**: 检查当前工作状态
- **!**: 切换到命令行模式，如 `!npm run dev`，执行结果会写入对话上下文
- **#**: 记忆模式，可写入项目级或用户级的长期记忆

## 使用建议

1. **项目开始时**: 使用 `/init` 让AI全面了解项目结构
2. **任务切换前**: 使用 `/clear` 清理上下文，避免干扰
3. **复杂任务**: 根据难度选择合适的思考模式命令
4. **上下文过长**: 使用 `/compact` 优化token使用
5. **需要执行系统命令**: 使用 `!` 前缀直接在对话中执行
