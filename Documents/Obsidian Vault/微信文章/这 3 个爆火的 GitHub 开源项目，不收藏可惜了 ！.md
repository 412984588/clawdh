# 这 3 个爆火的 GitHub 开源项目，不收藏可惜了 ！

嗨喽各位宝子！又到开源项目盘点的时间了，公众号每天都会送上3款GitHub 开源项目，供大家学习使用！

1

   

free-for-dev

给大家分享一个超实用的开源项目free-for-dev, 他收集了1100多个对开发者免费的服务和工具，拥有96000的star，是GitHub上最受欢迎的资源合集，这里面包含了很多云服务的免费额度，比如AWS服务器、Google cloud, 数据库方面，MongoDB、atlas给5GB免费存储，Fire base每天提供10万次数据操作，对个人项目来说完全够用了。开发工具这块也很给力，GitHub codespaces每月免费提供120小时的云端开发环境，Gitpod也有不少免费时长，还有Vs code的在线版，项目中还收集了很多实用的监控和API服务，比如Sentry可以帮你免费补获错误。Mailgun每月提供1万封免费邮件额度，推流有免费的短信API。对于学生来说，这个项目更是整理了超多专属福利，比如jetbrains全家桶、AWS学生专属资源、azure的100美元学生额度。网站按照不同类别整理了这些资源，包括云服务、数据库、开发工具、API和安全服务等，每个资源都清楚标明了免费额度和使用条件，做项目时特别方便。

https://github.com/ripienaar/free-for-dev

2

   

APILayer

这个项目可以让你5分钟接入3万+API。Github的star数已经达到了340K，APIlayer的try public apls for free项目，3万+API，覆盖天气、金融、体育等多领域，连postman官方都力荐，三步上手，找项目—试API—复制代码。5分钟接入。例如借助weatherstack API, 普通天气应用秒变专业级天气预报平台，这个API帮我把天气预报准确率提升了40%，Market stack API提供全球股市数据，让你的金融应用拥有蓬勃及分析能力，IP stack API实时解析访客位置、网站运营和安全防护的必备神器，Numverify API一键验证全球电话号码，杜绝虚假账号，注册转化率提升60%，Fixer API提供实时汇率，是跨境电商和金融应用的完美搭档。这些API可以让你的项目开发速度大幅提升。
https://github.com/public-apis/public-apis

3

   

PairDrop

作为一个完全开源的项目，Pairdrop已在github上获得了超过7000颗star，它让文件传输变得超级简单。打开浏览器不用安装任何软件，系统会自动给你分配一个随机名称，同一网络内的其他设备，一下就能看到多台设备，打开pairdrop后会自动互相发现，点一下目标设备就能开始传文件，方便的不得了。所有数据都是端到端加密的，安全有保障。想自己部署pairdrop也很容易，几个简单命令就能在本地运行起来，或者用docker容器也行，个人使用或者公司内网都没问题。数据不经过第三方，私密性更好。Pairdrop在各种场景下都能派上大用场，会议中快速分享文件，朋友聚会传照片都不在话下，不用注册账号，不用复杂配对，最重要的是你的数据只在设备之间直接传输，没有中间服务器，隐私安全有保障。

https://github.com/schlagmichdoch/PairDrop
