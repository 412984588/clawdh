# OpenCode复刻Claude Work开源OpenWork；智能AI提示词版本管理与共享

✨ 1: OpenWork

智能体工作流桌面应用




OpenWork是一个可扩展的开源“Claude Work”风格的知识工作者系统，旨在将代理式工作转化为直观的产品体验，而非复杂的命令行操作。其核心功能包括提供一个可视化的桌面应用程序，通过引导式工作流简化任务执行，支持用户选择工作区、启动任务、监控进度、按需批准权限，并复用已有的模板和技能。该系统具备高度扩展性，允许安装技能和OpenCode插件；提供审计功能，清晰记录操作过程；具有权限管理机制，对敏感操作进行授权控制；并且支持本地运行（作为OpenCode服务器的宿主模式）和连接远程OpenCode服务器（客户端模式）。此外，它还提供了会话管理、实时更新、执行计划可视化、可定制的任务筛选条件以及强大的技能管理器来安装和导入技能，并通过管理opencode.json文件来扩展OpenCode插件，从而为知识工作者提供了一个高效、安全且用户友好的自动化工作平台。

地址：https://github.com/different-ai/openwork

✨ 2: Prompt Manager

智能AI提示词版本管理与共享

Prompt Manager (PromptX)是一款致力于提升AI工作效率的生产力工具，以其极致现代感的UI/UX设计、智能版本管理和跨平台同步为核心特色。该项目提供全面的个人提示词管理功能，包括安全的JWT用户认证、高效的提示词CRUD操作、支持嵌套与树形结构的智能分类、全局搜索及高级标签系统，同时内置一键复制、收藏和智能分页功能。其核心亮点在于强大的版本管理系统，能够自动保存每次修改、直观地对比不同版本差异并支持一键恢复，确保用户灵感不失。此外，AI智能修改功能允许用户通过输入优化建议，由AI自动优化提示词。项目还创新性地引入了公共模板库，构建了一个活跃的提示词分享社区，用户可浏览、搜索、点赞、下载社区分享的优质提示词，并一键分享自己的内容，通过动态分类、智能排序及每日免审核额度保障内容质量。完善的评价系统支持星级评分、文字评论、作者回复、评论点赞、举报及敏感词过滤，并强制下载验证与单次评价限制以防刷评。为了提供实时的交互体验，PromptX通过WebSocket实现了实时通知系统，即时推送评论和回复信息。管理员则拥有专属的审计中心，可以高效管理公共提示词的审核、分类及评论，确保社区秩序。项目采用Python FastAPI后端与React前端技术栈，支持Docker部署并集成了Watchtower实现自动热更新，同时为未来的桌面和移动应用迁移做好了CORS配置准备。

地址：https://github.com/lwh8915/PromptX

✨ 3: vibe-blog

AI技术写作助手

vibe-blog是一个基于多Agent协作架构的AI技术长文博客生成工具，旨在大幅降低技术写作门槛，使复杂的技术知识转化为通俗易懂的科普文章。该项目通过一系列智能Agent（如研究员、规划师、作者、提问官、程序员、艺术家、评审员、组装员等）自动化完成从素材收集、深度调研、大纲规划、内容撰写、代码集成、智能配图（包括Mermaid图表、AI封面图、上下文感知配图和八种风格选择）到质量审核的全流程写作。其核心功能体现在支持多种文章类型（教程型、问题解决型、对比分析型）和不同长度（短文、中等、长文，长文模式提供更深度的追问和原理剖析），并特别包含vibe-reviewer模块进行多维度内容质量检测（涵盖准确性、逻辑性、完整性、可读性及中文可读性分析）、问题精准定位和报告导出。该工具还支持Markdown和长图等多种格式输出与实时预览，旨在有效解决传统技术写作耗时费力、配图困难、深度不足及受众单一等痛点，并规划发展成为技术知识共创平台和智能写作IDE。

地址：https://github.com/datawhalechina/vibe-blog

✨ 4: browser

AI 智能体浏览器控制

browser项目是一个专为AI代理设计的Chrome浏览器自动化工具，它旨在以最高的速度和token效率赋能AI代理进行网页操作。该工具提供了一套丰富的命令行接口，涵盖了从浏览器启动/停止、页面导航（打开、跳转、前进、后退、刷新）、标签页管理到页面交互（点击、输入、悬停、选择、滚动、等待）、内容提取（获取大纲、文本、HTML、查找元素、URL、标题）等核心功能。此外，它还支持截图、调试（控制台输出、网络请求、JavaScript评估）、会话状态（cookies、localStorage）及视口和用户代理的灵活配置。项目通过提供“技能文件”机制，简化了AI代理对这些能力的集成与利用，并允许用户使用标签、文本或CSS选择器进行元素定位，从而为AI代理提供了全面且高效的Web控制能力。

地址：https://github.com/camhahu/browser

✨ 5: DeepResearch Eval

深度研究任务生成与智能体评估框架

DeepResearch Eval 是一个专注于深度研究任务构建与智能体评估的自动化框架。该项目通过其核心的任务生成机制，利用基于用户画像的流程，创建真实且复杂的深度研究任务，并经过两阶段筛选确保任务需整合多源信息并进行外部检索。在评估方面，它提供了两项关键功能：一是自适应逐点质量评估，能够根据每个任务动态生成评价维度、标准和权重；二是主动事实核查，即使缺少引用，也能通过网络搜索自主提取并验证报告中的陈述，从而全面衡量深度研究系统的性能和事实准确性。

地址：https://github.com/Infinity-AILab/DeepResearchEval
