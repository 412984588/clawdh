# 杀疯了！3 个藏不住的 GitHub 开源项目，好用到哭

哈喽啊兄弟们，又到开源项目分享时间了。公众号每天都会送上3款GitHub 开源项目，供大家学习使用！




1



   

tunnelto

本地服务，不用服务器，公网直接访问今天这神器必藏tunnelto，本地服务公网访问无需服务器，这是rust写的本地隧道神器，不用买服务器，不用配置域名，一行行命令把本地服务变公网，可访问链接开发者最懂得痛调试支付回调第三方接口本地服务外网访问不了，上手贼简单，支持macOS/Windows/Linux, macOS系统只需要brew install agrinman/tap/tunnelto立马搞定，启动指定端口，秒出公网链接，复制就能用。喜欢赶紧收藏，防止找不到。

https://github.com/agrinman/tunnelto




2

   

ChatGPT-Shortcut

这是一个市面上最全的AI提示词开源宝藏库，在这里你可以看到各个领域的各种专业提示词，最夸张的是它的社区功能，你可以在这里看到各路大神上传的提示词，甚至我们还能给提示词打分，而且操作非常简单，你只需直接复制就能使用，非常方便，让你AI更聪明。

https://github.com/rockbenben/ChatGPT-Shortcut




3

   

happy


兄弟们，这开源项目真的太牛了，它的名字叫happy，顾名思义就是开心的意思。可以看到，目前已经收获了5.4K的star。这是一个完全可以在手机上使用codex和cloud code的项目，也就是说，你现在通过这个开源项目，可以完全的在手机上去操控你的cloud code进行编程。在任何地方都可以使用cloud code和codex，，完美的拼齐了在手机上使用终端编程工具的遗憾。无论是我们的iOS还是安卓，还是这种Linux，都支持去使用这个工具。如果你需要使用这个cloud code的话，你只需要在那个手机上执行一个happy，然后这时候就可以调取cloud code或者codex。它的原理也非常简单，实际上就是通过happy这个代替cloud code, 通过这个命令呢，去来将你远程的去控制你的这个程序，实际上整个项目还是非常的有意思的，如果这个视频对你有帮助的话，点不还点个关注，我们下期再见。

https://github.com/slopus/happy
