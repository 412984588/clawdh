Mermaid

Mermaid 是一个基于 Javascript 的图表绘制工具，通过解析类 Markdown 的文本语法来实现图表的创建和动态修改。Mermaid 诞生的主要目的是让文档的更新能够及时跟上开发进度。

https://github.com/mermaid-js/mermaid?tab=readme-ov-file

2

   

简历生成器

Reactive Resume是一个免费的开源简历生成器，可以简化创建、更新和共享简历的过程。该平台没有任何用户跟踪或广告，您的隐私是重中之重。该平台非常用户友好，如果您想完全拥有自己的数据可以在不到 30 秒的时间内自行托管。
它提供多种语言版本，并提供实时编辑、多种模板、拖放自定义以及与OpenAl集成以增强您的写作能力等功能。
你可以将简历的个性化链接分享给潜在的雇主，跟踪简历的浏览量或下载量，还可以通过拖拽的方式定制页面布局。平台还支持各种字体选项，并提供了数十种模板可供选择。是的，甚至有一个黑暗的模式，更舒适的观看体验。
https://github.com/AmruthPillai/Reactive-Resume

3

   

RoomGPT

一个AI驱动的室内设计工具，通过上传房间照片生成设计建议。这是 RoomGPT.io的上一个开源版本(一个付费的 SaaS 产品)。这是roomGPT的第一个版本，没有授权、支付或其他附加功能，易于克隆、部署和操作。

https://github.com/Nutlope/roomGPT
