# 开源的一站式 Agent 聊天机器人平台，可接入主流即时通讯软件，为个人、开发者和团队打造可靠、可扩展的对话式智能基础设施

AstrBot


“

一个开源的一站式 Agent 聊天机器人平台，可接入主流即时通讯软件，为个人、开发者和团队打造可靠、可扩展的对话式智能基础设施。无论是个人 AI 伙伴、智能客服、自动化助手，还是企业知识库，AstrBot 都能在你的即时通讯软件平台的工作流中快速构建生产可用的 AI 应用。

Github地址

https://github.com/AstrBotDevs/AstrBot

官网地址

https://docs.astrbot.app/

项目特点


免费开源：完全免费且开源，用户可以自由使用和修改代码。
多平台支持：支持 QQ、企业微信、飞书、钉钉、微信公众号、Telegram、Slack 等多种即时通讯平台。
强大的插件系统：拥有近 800 个插件，可一键安装扩展功能。
多语言支持：提供国际化（i18n）支持，有多种语言版本的文档。
WebUI 支持：具备 WebUI 界面，方便用户操作和管理。
AI 大模型对话：支持多种 AI 大模型服务，如 OpenAI、Gemini、Anthropic 等，还支持多模态、Agent、MCP、知识库、人格设定等功能。
语音转文本和文本转语音服务：支持多种语音转文本和文本转语音服务，如 OpenAI Whisper、SenseVoice、OpenAI TTS 等。
支持的消息平台


官方维护：QQ（官方平台 & OneBot）、Telegram、企微应用 & 企微智能机器人、微信客服 & 微信公众号、飞书、钉钉、Slack、Discord、Satori、Misskey、Whatsapp（将支持）、LINE（将支持）。
社区维护：KOOK、VoceChat、Bilibili 私信、wxauto。
部署方式



Docker 部署（推荐）：通过 Docker / Docker Compose 方式部署，具体操作可参考官方文档 使用 Docker 部署 AstrBot。

uv 部署：使用命令 uvx astrbot 进行部署。

宝塔面板部署：AstrBot 已上架至宝塔面板，部署方法参考官方文档 宝塔面板部署。

1Panel 部署：AstrBot 已由 1Panel 官方上架至 1Panel 面板，部署方法参考官方文档 1Panel 部署。

雨云平台部署：AstrBot 已由雨云官方上架至云应用平台，可一键部署，具体链接为 雨云平台。

Replit 部署：社区贡献的部署方式，链接为 Replit 部署。

Windows 一键安装器部署：参考官方文档 使用 Windows 一键安装器部署 AstrBot。

CasaOS 部署：社区贡献的部署方式，文档链接为 CasaOS 部署。

手动部署：首先安装 uv，然后通过 Git Clone 安装 AstrBot，命令为：

pip install uv
git clone https://github.com/AstrBotDevs/AstrBot && cd AstrBot
uv run main.py


或者参考官方文档 通过源码部署 AstrBot。
