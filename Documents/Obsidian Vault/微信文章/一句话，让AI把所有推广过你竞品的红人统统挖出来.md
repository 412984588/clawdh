# 一句话，让AI把所有推广过你竞品的红人统统挖出来

> 作者: 刘小排r | 发布日期: 2025年12月5日 12:45

哈喽，大家好，我是刘小排。
我常常跟朋友说，流程即代码：但凡有SOP，就能被自动化。
正好今天有个简单的案例，分享给大家。


一个案例：找出推荐过某产品的所有达人


如果你也做AI SaaS产品的联盟营销，那你很可能想要这样一种工具： 自动化把推广过你某个竞品的所有网红找出来，包括联系方式、推广效果、推广素材、优惠码。
如下图所示。我节选了生成报告的一部分截图。


以前，这件事怎么做？
以前，这样的事情一般是靠实习生做。


公司不断积累自己的SOP，比如： 怎么搜索红人、怎么筛选红人、怎么寻找红人联系方式、怎么联系红人、怎么撰写邮件等等。


这样的SOP，对具体操作人员的技能要求不高。


因此，公司理性的选择应该是
用精兵强将不断优化升级SOP
用实习生和初级员工不断执行SOP，放大结果


用AI做的解题思路
如何用AI呢？
我的解题思路是： 用Claude Skills
报告虽然复杂，但是生成过程有SOP，有详细的步骤。 
我一向认为 —— 只要有SOP的工作内容，就适合用Claude Skills。  （如果想要了解更多，详见 Claude Skills 不就是把提示词存个文件夹吗？ ）
下面的截图，展示了最终成品的Claude Skill
以后我只需要输入“ 帮我找所有推荐过 [产品名] 的 YouTuber 和红人"
AI就能够自动完成所有的预设SOP、完成上面的报告。


具体怎么做的？


只需要三步： 梳理SOP - 提供配套工具 - 编写Claude Skill


1. 梳理SOP。 如上图所示，我们的SOP一共有6个阶段工作。你可以放大看图。
在这案例里，核心的SOP流程是
阶段
	
内容

Phase 1	
产品情报收集 - 了解产品、定价、竞品

Phase 2	
多平台发现 - YouTube 20+ 搜索变体、Twitter 10+ 搜索变体

Phase 3	
深度数据充实 - 频道统计、联系方式、联盟指标

Phase 4	
验证去重 - 数据清洗、质量评分

Phase 5	
分层组织 - Mega/Major/Mid/Micro/Nano 分级

Phase 6	
报告生成 - 500+ 行完整报告


2. 提供需要用到的工具。 有的工具是现编写的Python脚本、有的是MCP、有的工具是某个现成的脚本或工具、有的是API …… 
在这个案例里，至少我们需要：搜索Youtube的工具、搜索Twitter的工具、查找Linkedin的工具、分析竞品的工具、数据清洗的工具、打分的工具、查找邮箱的工具等等……


3. 编写Claude Skill。


真正值钱的工作，是第1步


有少许技术含量的工作，是第2步，不过也不算什么难事。


至于第3步，只需要告诉Claude Code “我需要把刚才的对话变成skill” 就完成了。


你可能想问




既然初级员工的工作，很容易被自动化，以后，公司的壁垒是啥？


- 壁垒是你的SOP！ 千万不要把你的工作流、SOP，轻易分享给同行。因为：流程即代码； 一旦别人拥有你的SOP，他就能在AI的帮助下，以同等的质量、完美复制你的每一次动作。




既然初级员工的工作，很容易被自动化，以后，个人如何发展？


- 你要去做那个发明和优化SOP的人，不要去做执行SOP的人




总结



流程即代码：有 SOP 的地方，就有自动化的空间。

对公司来说：SOP 就是护城河，别轻易外流。

对个人来说：写 SOP、设计流程的人，才是 AI 时代最值钱的人。

Claude Skills，本质上只是：
帮你把脑子里的 SOP，变成一套可执行的“自动机器人”。
