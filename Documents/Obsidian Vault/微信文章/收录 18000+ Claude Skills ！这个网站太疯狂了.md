收录 18000+ Claude Skills ！这个网站太疯狂了
前言

自从 Claude Skills 功能上线以来，社区里涌现出了大量优质的 Skills。但问题来了——这些散落在 GitHub 各个角落的神器，找起来实在太费劲了！

于是，有大佬站出来了。

他把 GitHub 上能找到的 Claude Skills 全部爬了下来，建了一个可搜索的目录网站。目前已收录 18000+ 个 Skills，堪称 Claude 生态的「神器百科全书」！

网站地址：skillsmp.com

一、核心功能一览
1.1 分类浏览

Skills 按用途分门别类，主要类别包括：

• 开发工具：代码生成、调试助手、重构工具
• 文档生成：README 生成器、API 文档助手
• AI 增强：提示词优化、输出格式化
• 数据分析：CSV 处理、数据可视化
• ……以及更多
1.2 关键词搜索

输入关键词，秒速定位你需要的 Skill。再也不用在 GitHub 里大海捞针了！

1.3 预览 SKILL.md

下载之前先看看这个 Skill 到底能干啥。省时省力，避免踩坑。

1.4 一键下载

看中哪个，直接下载 ZIP 包。简单粗暴，效率拉满。

1.5 CLI 安装

如果项目包含 marketplace.json，可以直接在 Claude Code 中通过命令行安装。真·无缝集成！

二、未来规划

作者透露了几个正在考虑的方向：

功能
	
说明

评分系统	
让用户给 Skill 打分，筛选优质内容

教程中心	
为热门 Skills 制作视频/文字教程

提交入口	
允许创作者直接提交自己的 Skill
三、写在最后

虽然网站目前还比较基础，但这个方向绝对是社区刚需。随着 Claude Skills 生态的不断壮大，这样的聚合平台只会越来越有价值。

收藏这个网站，以后找 Skill 就靠它了！

🔗 网站地址：https://skillsmp.com
