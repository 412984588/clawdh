deepdarkCTI 是一个收集来自暗网和深网的网络威胁情报源的工具。简单讲，它帮助安全研究人员快速获取暗网中的黑客活动、数据泄露等网络安全威胁信息。适用人群：网络安全研究人员、CTF选手、威胁情报分析人员。

项目概述

deepdarkCTI项目致力于收集来自深网和暗网的网络威胁情报源，官方网站为https://www.deepdarkcti.com 。其目标是收集深网和暗网中对网络威胁情报（CTI）有价值的信息源。

核心功能
情报收集：聚焦于深网和暗网相关的开源情报（OSINT）源，对以下来源的情报信息进行监测：
Telegram 频道、群组和聊天
Discord 频道
勒索软件团伙网站
与网络犯罪活动和数据泄露相关的论坛
市场
漏洞数据库

方法指导：在methods文件中，描述了各种搜索和分析情报源的技术。
网络威胁情报相关知识
定义：网络威胁情报（CTI）是指收集和分析有关威胁和对手的信息，并总结出模式，以便为应对各种网络攻击的准备、预防和响应行动做出明智决策。它涉及收集、研究和分析网络威胁领域的趋势和技术发展，通常以妥协指标（IoCs）或威胁源的形式呈现，为组织独特的威胁态势提供基于证据的知识。
分析依据：基于意图、能力和机会三元组进行分析，专家可通过研究该三元组，对组织面临的现有或新兴威胁做出明智的战略、运营和战术决策。
类型：
战略情报：提供有关网络安全态势、威胁及其对业务影响的高级信息。
战术情报：提供与威胁行为者实施攻击所使用的战术、技术和程序（TTPs）相关的信息。
运营情报：提供有关针对组织的特定威胁的信息。
典型情报源：包括开源情报（OSINT）、人力情报、反情报和内部情报。
项目优势
专注特定领域：聚焦于深网和暗网的 OSINT 源，能获取一些常规渠道难以获得的网络威胁情报。
交流与协作：提供 Telegram 群组供业内人士交流，有助于分享经验和发现新的情报源。
方法指导：在methods文件中提供搜索和分析技术，为情报收集和分析工作提供了指导。
应用场景
企业安全：企业可利用该项目收集的情报，了解自身面临的网络威胁，提前做好防范措施，保护企业的信息资产安全。
安全研究：安全研究人员可通过该项目获取深网和暗网的情报，进行网络威胁的研究和分析，推动网络安全技术的发展。
执法部门：执法部门可借助这些情报，打击网络犯罪活动，维护网络空间的安全和秩序。
告别向量数据库！PageIndex推理式RAG系统实现98.7%专业文档分析准确率

项目概述

PageIndex是VectifyAI团队开发的一个推理式检索增强生成（RAG）系统，灵感源自AlphaGo。它旨在解决传统向量数据库在处理长专业文档时检索准确性不足的问题，通过构建文档的树状索引并基于此进行推理检索，模拟人类专家从长文档中提取知识的方式。

核心功能
推理式检索：传统向量检索基于语义相似性，而PageIndex关注真正的相关性，通过推理实现更精准的检索。它分两步进行检索，先为文档生成“目录”式的树状结构索引，再通过树搜索进行推理式检索。
树状结构生成：能将长PDF文档转化为类似“目录”的语义树状结构，适用于金融报告、学术教材、法律或技术手册等超过大语言模型（LLM）上下文限制的文档。
多格式支持：除了PDF，还支持对Markdown文件生成树状结构，但Markdown文件需格式正确，若由PDF或HTML转换而来，建议先使用PageIndex OCR转换后再处理。
主要优势
无需向量：不依赖向量数据库，利用文档结构和LLM推理进行检索。
无需分块：文档按自然章节组织，而非人为分块。
类人检索：模拟人类专家在复杂文档中导航和提取知识的方式。
检索过程透明：基于推理的检索，告别近似向量搜索。
性能表现

由PageIndex驱动的推理式RAG系统Mafin 2.5在FinanceBench基准测试中达到了98.7%的准确率，在专业文档分析方面展现了领先性能。

部署选项
本地部署：可通过此开源仓库进行本地部署，运行相关脚本即可。
云服务：提供云服务，可通过Agent、Dashboard或API快速体验，无需额外设置。
快速上手

提供了简单的 Vectorless RAG Notebook，可在Colab中直接运行体验推理式RAG流程。

包使用方法
安装依赖：运行 pip3 install --upgrade -r requirements.txt 安装所需依赖。
设置API密钥：在根目录创建 .env 文件，并添加OpenAI API密钥 CHATGPT_API_KEY=your_openai_key_here。
生成树状结构：运行 python3 run_pageindex.py --pdf_path /path/to/your/document.pdf 为PDF文档生成树状结构，还可通过可选参数进行自定义设置；对于Markdown文件，使用 python3 run_pageindex.py --md_path /path/to/your/document.md 生成。
高级功能
PageIndex OCR：针对复杂PDF文档，PageIndex OCR能更好地保留文档的全局结构和层次信息，在识别文档层次和语义关系方面优于其他OCR工具，可通过Dashboard体验或通过API集成。
