# 告别排版焦虑！这个开源项目让 ClaudeCode 帮你全自动发公众号

> 作者: 恶人笔记 | 发布日期: 2026年1月2日 17:00
> 原文链接: [点击查看](https://mp.weixin.qq.com/s/ANqWa240odgkBaDzE1cuSg)

---

做过公众号的朋友都知道,写文章最痛苦的往往不是"写"本身,而是排版和发布。你辛辛苦苦在 Notion 或 Markdown 编辑器里写好了干货,复制到公众号后台,格式全乱了。调字体、调间距、插图片、居中对齐……这些琐碎的机械劳动,每次都要耗费半小时甚至更久。

如果我告诉你,只要对着 AI 说一句话,文章就能自动写好、自动排好漂亮的版,并且自动上传到公众号草稿箱,你信吗?

今天要分享的这个 GitHub 开源项目BND-1/wechat_article_skills,就是为了解决这个痛点而生的。它是基于 Anthropic 公司的Claude Skills打造的一套全自动化工作流。

01它是做什么的?

简单来说,它是一个给 Claude(目前最聪明的 AI 模型之一)安装的"外挂手"。

以往我们用 AI,只是让它生成文字,然后我们自己去复制粘贴。但有了这个项目,Claude 就不再只是一个聊天机器人,它变成了一个全能的公众号运营助理。

核心功能就三点,点点直击痛点:

✍️ 自动生成文章

你只需要给出一个大概的想法,比如:"帮我写一篇介绍 DeepSeek 模型的科普文章,风格要幽默一点。" Claude 就会利用其强大的逻辑能力,为你生成一篇结构清晰、内容详实的草稿。

🎨 自动专业排版(重磅功能!)

这是最让人感动的时刻。工具会将生成的 Markdown 文本,自动转换为微信公众号专用的 HTML 格式。

不仅是转格式:
它会把代码块渲染成漂亮的深色 IDE 风格;
不仅是文字:
它可以自动处理标题、引用、列表的样式,直接生成类似"黑客派"或"壹伴"那种精美的排版效果;
告别 CSS 噩梦:
你再也不用手动去调行间距和字体大小了。
📤 自动发布到草稿箱

排版好的文章,不需要你手动复制。这个工具会调用微信接口,直接把文章推送到你的公众号后台草稿箱。你打开公众号后台,文章已经安静地躺在那里了,连封面图都配置好了(如果配置了相关能力)。

02安全吗?靠谱吗?

很多朋友听到"自动发布"会担心:万一 AI 乱发东西怎么办?

这点作者考虑得很周到。这个项目的终点是"草稿箱" (Draft Box),而不是直接群发。

🤖自动化完成脏活累活:AI 帮你搞定写稿、排版、上传。

👤人类掌握最终决定权:你只需要最后登录后台,扫一眼草稿,确认没问题,点一下"群发"即可。

这就是完美的"人机协作" (Human-in-the-Loop)模式:AI 负责效率,你负责把关。

03如何开始?

这是一个开源项目,你可以免费获取和使用。

项目地址:
👉 https://github.com/BND-1/wechat_article_skills

你需要准备什么?

一个 Claude 账号(配合 Claude Code 或其它支持 MCP/Skills 的环境)。
一个微信公众号(订阅号或服务号均可)。
一点点 Python 基础(用来配置环境),或者直接将项目网址给到Claude Code,让他帮你安装好这个Skills。
04为什么推荐它?

市面上有很多公众号排版工具,但它们大多是"辅助工具",你还得自己动鼠标。

而 wechat_article_skills 代表的是"代理 (Agent)"的未来——你只管下指令,AI 负责全自动执行。

如果你是技术博主、独立开发者,或者是想从繁琐运营中解脱出来的创作者,这个项目绝对值得你 Star ⭐ 一下!

🚀开始你的自动化之旅
让 AI 成为你的公众号运营助理,把时间花在创作上,而不是排版上!




 最后的最后

其实上面的文章内容就是使用这个Skills自动美化排版并发布到公众号后台草稿箱的。虽然还有点小瑕疵，比如处理后的排版里部分中英文字符标点混乱的问题。

但重要的是项目作者已经把整个流程跑通了，我们就可以在这个项目上做更多的想象了。
