# 详解 & 实测 VoxCPM 1.5 ：Mac/Win 都能飞快跑

> 作者: kate人不错 | 发布日期: 2025年12月11日 02:21
> 原文链接: [点击查看](https://mp.weixin.qq.com/s/2wqLhk3DRnGj7fvUB1HQvA)

大家好，我是 Kate。

之前我介绍过 VoxCPM-0.5B，最近这个项目迎来了重大更新——升级到了 1.5 版本。这里先说明一下，"1.5"指的是版本号，不是参数量。新版本在音质、效率和稳定性上都有明显提升，还原生支持了全参数微调和 LoRA 微调。

更让我惊喜的是，这次在 Mac 上的体验终于流畅了！以前生成 10 秒音频要等好几分钟，现在只要十秒就能出结果。

今天就结合我的实测，系统聊聊这个版本。

https://github.com/OpenBMB/VoxCPM




VoxCPM 是什么？

VoxCPM 是基于 180 万小时双语语料训练的语音合成模型，主要支持中文和英文。它最大的特点是零样本语音克隆——只需要一小段参考音频，就能克隆出说话人的声音，不需要为每个人重新训练模型。







更厉害的是，它不仅能学到音色，还能捕捉说话的口音、情感和节奏。所以生成的语音听起来比较自然，不容易出现那种"机械感"的合成腔调。

在性能方面，它在 4090 显卡上生成速度非常快；从 9 月底开始，官方也支持了 Mac 的 MPS 加速，这对 Mac 用户来说太友好了。

1.5 版本升级了什么？

和 0.5B 相比，1.5 版本有三个最明显的升级。

第一，采样率大幅提升。 新版本采样率达到了 44.1kHz，高频细节更丰富。人声中的气息感、齿音、尾音这些细微部分刻画得更好，零样本克隆的精细程度也随之提升。




第二，效率几乎翻倍。 Token 处理效率从原来的 12.5Hz 降到了 6.25Hz。同样长度的音频，现在只需要处理一半的 Token 数量，推理算力需求几乎减半。所以建议大家直接用 1.5 版本的 800M 模型，别再纠结旧版本了。




第三，稳定性更好。 目前单次非流式生成的安全上限大概在 5 分钟左右，实用区间推荐控制在 3-5 分钟，这样连贯度和稳定性都比较有保障。




安装和使用体验

这次的安装流程比之前简单很多。装好依赖后，可以通过 Python 脚本或命令行使用，还提供了非常方便的 WebUI 界面。

您可以通过运行 python app.py 启动 WebUI 界面，该界面允许您执行语音克隆和语音创建。

WebUI 分为两个主要界面：语音生成界面用来输入文本、上传参考音频并生成语音；微调训练界面用来做全参数或 LoRA 训练。整个流程基本都能在浏览器里完成，对普通创作者非常友好。







几个实用功能详解
Text Normalization：数字和标点的自动处理

WebUI 里有个 Text Normalization 开关。打开时，输入"123"这样的数字，模型会自动调用 WeTextProcessing 库处理成正常读法。




如果关闭这个功能，你就可以精细控制发音，比如用带声调的拼音标注，让模型严格按你的要求来读。这种模式适合对发音有苛刻要求的场景。

参考音频的"学习"能力

上传参考音频时，如果原音频有独特的音色、说话风格，甚至带有背景声，这些特征都会被学过去。模型不仅在学"谁在说"，还在学"以什么样的氛围在说"。




如果你想要干净的人声，可以打开 Prompt Speech Enhancement 功能，模型会偏向输出去噪后的版本。

不上传参考音频也能用

这是 VoxCPM 一个很有意思的能力：即使不提供参考音频，它也能根据文本自动判断合适的说话风格。

这背后借助的是 MiniCPM-4 的文本理解能力。给它一段童话故事，它会用温柔亲切的语气来讲；换成促销广告，它又会自动切换成有冲击力的推销腔。

CFG Value 参数调节

如果生成的声音听起来紧绷或奇怪，可以尝试降低 CFG Value；想要更清晰聚焦，则适当提高。短句一般可以调高一点，追求自然度则找个中间值。




实测效果：五种场景体验

为了直观感受 1.5 版本的表现，我测试了几种完全不同的场景。

童话故事： 不上传参考音频，只给一段童话文本。大概十秒就生成好了，在省略号处会自然停顿，节奏拿捏得很准，音色也很贴合儿童向的旁白风格。




促销广告： 换了一段"清仓大甩卖"式广告脚本，不到十秒完成。冲击力很强，节奏紧凑、语气夸张，和我们在短视频里听到的广告声线很像。




相声腔读新闻： 上传一段相声演员的音频作为参考，让它读美联储降息的新闻稿。"3.5% 至 3.75%"这样的数字都读得很准确，同时保持了相声式的语气，非常有趣。

姜文式情绪表达： 上传一小段姜文的原声，让它读同一段新闻稿。生成的声音明显速度更快、情绪更激昂，模型把参考音频里的情绪模式也学了过去，不只是音色。




《甄嬛传》片段： 上传一段台词音频，自动识别的文字有些错误，手动修正后生成的音频在音色和情绪上都非常接近原版，那种略带威严的说话方式被很好地复刻出来。

综合来看，无论是童话、广告、新闻还是影视片段，VoxCPM 1.5 在风格迁移和情绪模仿方面都相当稳定。

微调能力

VoxCPM 1.5 同时支持全参数微调和 LoRA 高效微调。




关于 LoRA 微调需要多少数据，官方做过实验：用 5 分钟和 10 分钟的语音数据，默认配置训练 2000 步，都能收敛到比较统一的音色。也就是说，做 Demo 级别的展示，10-30 分钟的高质量数据就够了。




从 0.5B 升级到 1.5

如果你之前装过 VoxCPM-0.5B，升级也不复杂。我这次借助 AI 编程工具完成了整个过程，包括拉取最新代码、保留自定义配置、用 uv 重新同步依赖等步骤。









在 AI 工具辅助下，整个迁移过程非常顺畅，服务启动后访问 localhost:7860 就能看到新版 WebUI 了。

总结

VoxCPM 1.5 在采样率、推理效率和稳定性上都有实打实的提升，配合完善的 WebUI、微调能力，以及 Mac 上也能流畅运行的性能表现，无论是做内容创作、拟人播报，还是定制声音实验，都非常值得尝试。

如果你之前还在用 0.5B 版本，可以考虑切到 1.5 版本体验一下。
