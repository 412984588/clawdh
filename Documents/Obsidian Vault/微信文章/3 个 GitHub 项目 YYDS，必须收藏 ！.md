OpenStock

OpenStock是一个开源的替代方案，用于昂贵的市场平台。可追踪实时价格、设置个性化提醒并探索详细的公司洞察--公开构建，为所有人服务，永久免费。注意:OpenStock是由社区构建的，不是经纪公司。市场数据可能因提供商规则和您的配置而延迟。此处没有任何财务建议。

pocketbase

这个后端开发神器在 GitHub 上获得了 5.2 万个星标，提供了完整的后端解决方案，涵盖数据库、用户系统、文件存储、管理后台等功能，支持单文件部署，双击即可启动全部功能，极大简化了开发流程。传统方案需要安装 MySQL、配置 Redis、搭建文件服务器，而它的设计让你无需这些繁琐步骤，所有功能都一站式提供。实时推送功能尤为强大，聊天应用、协作工具、数据监控等功能，让用户操作瞬间同步到所有设备，完全支持 Websocket 和消息队列。此外，它还内置了零代码拖拽式编辑、企业级 REST API 自动生成、JWT 认证、OAuth 登录等功能，且基于 Go 语言内核，性能极其优越，能轻松应对数千并发请求。如果你还在为后端开发烦恼，这个项目绝对不容错过。

https://github.com/pocketbase/pocketbase

3

   

maxun

无需编写任何代码，任何人都可以在两分钟内训练机器人，自动抓取网页数据。全球用户通过 Maxun 已节省超过 23000 小时，提取了超过 200 万行数据，项目在 GitHub 上获得了 12000 个星标。Maxun 的部署非常简便，支持两种方式：一是通过 Docker 使用几个简单命令一键部署；二是通过 GitHub 克隆代码进行手动安装，系统需要 Node.js、PostgreSQL、Minio 和 Redis 的支持


https://github.com/getmaxun/maxun
