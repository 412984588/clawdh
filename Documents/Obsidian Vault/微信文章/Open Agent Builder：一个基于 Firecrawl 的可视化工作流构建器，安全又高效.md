# Open Agent Builder：一个基于 Firecrawl 的可视化工作流构建器，安全又高效

> 作者: Halo咯咯 | 发布日期: 2025年10月21日 19:50
> 原文链接: [点击查看](https://mp.weixin.qq.com/s/3S07p8kQw7-pn3dDwH3xww)

---

Open Agent Builder 是一个基于 Firecrawl 的可视化工作流构建器，旨在帮助用户通过无代码界面构建、测试和部署 AI 代理工作流。以下是其核心要点：

功能特点：

拖拽式界面：用户可以通过拖拽组件的方式构建工作流，实时执行并查看结果。
核心节点类型：包含 8 种核心节点类型，涵盖从开始到结束的全流程构建。
模板库：提供丰富的模板，方便用户快速上手。
MCP 协议支持：支持多种工具集成，便于扩展。
Firecrawl 集成：强大的网络爬虫功能，支持数据提取和自动化任务。
企业级特性：支持 LangGraph 执行引擎、Clerk 认证、Convex 数据库等，确保安全性和可靠性。

技术栈：

前端：采用 Next.js 16 和 Tailwind CSS，提供流畅的用户体验。
后端：使用 LangGraph 和 Convex，支持多种 AI 提供商，如 Anthropic、OpenAI 和 Groq。
安全执行：通过 E2B 实现沙箱代码执行，确保安全。

安装与部署：

先决条件：需要 Node.js 18+、Firecrawl API 密钥、Convex 和 Clerk 账户。
安装步骤：提供详细的安装指南，包括 Convex 数据库和 Clerk 认证的配置。
部署：支持 Vercel 部署，方便用户快速上线。

使用场景：

网络爬虫：支持数据提取和自动化任务。
多步 AI 代理流程：适用于复杂的自动化工作流。
自动化研究和内容生成：支持人类参与的敏感操作审批，确保流程的可靠性。
