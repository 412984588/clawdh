# 刷爆朋友圈！这 3 个 GitHub 开源项目，堪称效率天花板

> 作者: 软件冢 | 发布日期: 2026年1月7日 22:53

哈喽啊兄弟们，又到开源项目分享时间了。公众号每天都会送上3款GitHub 开源项目，供大家学习使用！




1



   

web-check

这是一个专门分析网站的开源工具。平时上网你访问的网站安全吗？用它扫一下，能看到服务器在哪个国家，证书安不安全，DNS配置对不对，用了什么技术，有没有恶意代码，会不会是钓鱼网站，网站的历史变化也能查到。平时遇到不认识的网站，扫一下看看底细，想学别人的技术可以研究研究架构，如果是做运维工作的，用来定期检查也很方便。这工具在github上挺受欢迎，已经拿下了28Kstar了，代码完全开源，支持docker一键部署，感兴趣的可以去试试。

https://github.com/Lissy93/web-check




2

   

galaxy

前端封神级开源项目，3000+免费UI组件，复制粘贴直接用，前端开发效率翻倍，Galaxy这是UIverse官方维护的保障仓库。3000多个全球设计师贡献的优质UI组件，你小到按钮、复选框、开关、卡片，你能想到的UI元素这里都有，而且官网支持可视化预览和标签筛选，想要什么风格直接搜，看中了再去仓库复制代码，找组件的效率直接拉满，重点来了，所有组件都基于MIT开源协议，商用、修改、分发都完全自由，这一点对开发者来说太重要了，一使用起来非常简单，支持我们导出Vue、HTML、React立马快速使用，无需下载，直接复制粘贴就行，开源免费，还在持续更新，谁用谁香，感兴趣的赶紧收藏，防止找不到。

https://github.com/uiverse-io/galaxy




3

   

pocketbase

这是一款无需编码的后端快速开发神器，在GitHub上斩获了5.5W星星。它仅是一个文件，短短几分钟就能搞定后端开发，还能适配多个主流前端框架。你还能通过Web界面直接管理数据，一键就能完成增删改查操作，系统会自动生成Restful API接口。另外，它支持实时订阅、身份认证和本地存储等功能，无需单独部署数据库服务器，只需配置好约定缓存和硬件反向代理就能运行。恭喜你找到这个好用的后端工具。


https://github.com/pocketbase/pocketbase
