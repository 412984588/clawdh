# 你的 Mac 变慢了吗？这就派只“鼹鼠”去挖出垃圾

> 作者: 面条实验室 | 发布日期: 2025年12月9日 08:11
> 原文链接: [点击查看](https://mp.weixin.qq.com/s/sMJatZsD00j8KLoNk5c53w)

---

你的 Mac 用久了，是不是总感觉没以前顺滑了？



硬盘空间莫名其妙被占满，想删软件却总删不干净，系统缓存像滚雪球一样越积越多。市面上的清理工具要么收费昂贵，要么界面花哨得让人头晕。

今天，我想给你推荐一款刚刚开源的、小而美的命令行工具 —— Mole。



GitHub
它是谁？

Mole（鼹鼠）是由博主 @tw93 开发的一款 macOS 全能工具箱。

正如它的名字一样，它像一只勤劳的鼹鼠，深入你的系统底层，帮你挖掘并清理那些深埋的“垃圾”，让你的 Mac 重新焕发活力。它把系统清理、软件卸载、性能监控等功能，全都塞进了一个小巧的二进制文件里。

Mole
它能为你做什么？

Mole 不仅免费开源，而且功能非常实用。

1. 深度系统清理

还在手动翻文件夹找缓存吗？Mole 能一键扫描并清理系统缓存、日志文件、浏览器残留以及开发者工具产生的垃圾。释放空间，就在一瞬间。

深度系统清理
2. 智能应用卸载

直接把 App 拖进垃圾桶通常是删不干净的。Mole 的卸载功能类似 AppCleaner，它不仅能删除应用本身，还能顺藤摸瓜，把相关的偏好设置、启动项等残留文件一锅端。

智能应用卸载
3. 磁盘空间分析

硬盘红了却不知道大文件藏在哪？Mole 提供了一个可视化的磁盘分析器，帮你快速揪出占用空间的“大块头”，无论是高清电影还是遗忘的备份，都无处遁形。

磁盘空间分析
4. 实时状态监控

不需要安装繁重的监控软件，Mole 自带一个清爽的终端仪表盘。CPU、内存、网络、电池状态，一眼就能看清你的 Mac 此刻是否“压力山大”。

实时状态监控
如何开始使用？

Mole 的安装非常简单，不需要复杂的配置。

打开你的终端，复制并粘贴下面这行命令：

curl -fsSL https://raw.githubusercontent.com/tw93/mole/main/install.sh | bash


或者，如果你是 Homebrew 用户，也可以这样安装：

brew install tw93/tap/mole


安装完成后，只需要在终端输入 mole，你就能召唤这只神奇的“鼹鼠”了。

在这个软件越来越臃肿的时代，Mole 给我们带来了一股清流。它没有多余的广告，没有复杂的后台进程，只有简单、高效的工具体验。

如果你也是一名追求效率的 Mac 用户，不妨试试看。

项目地址： https://github.com/tw93/Mole
