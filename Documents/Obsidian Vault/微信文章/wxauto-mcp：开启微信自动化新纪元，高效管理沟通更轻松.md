# wxauto-mcp：开启微信自动化新纪元，高效管理沟通更轻松

> 作者: 热河逐梦码农哥 | 发布日期: 2025年10月27日 20:30

在数字化办公日益普及的今天，微信早已不再只是日常聊天工具，更成为工作沟通、信息传递的重要载体。但当需要同时管理多个微信账号，日复一日手动回复消息、定时发送内容时，时间和精力的消耗往往让人倍感压力。而今天要为大家介绍的 wxauto-mcp，正是一款能帮你摆脱这些繁琐操作的微信自动化工具 —— 它基于 wxauto 库构建，以 MCP 服务器的形式，为微信消息的发送与接收提供了强大的自动化支持，开启了微信高效管理的新纪元。

一、适用环境与版本要求

在开始使用 wxauto-mcp 之前，首先需要确认你的设备环境是否符合要求。目前，这款工具仅适用于 Windows 系统，且对微信版本有特定限制，具体要求如下：

请务必在安装前检查你的系统和软件版本，避免因版本不兼容导致工具无法正常运行。

三、安装与配置步骤详解

想要使用 wxauto-mcp，只需按照以下步骤完成安装和配置，即使是对技术不太熟悉的用户也能轻松操作。

第一步：安装 uv

uv 是一款高效的 Python 包管理工具，wxauto-mcp 的运行需要依赖它。打开终端，输入以下命令并运行：

"c:\program files\python\python.exe" -m pip install --upgrade pip

当终端中出现包含“Licensed under the MIT license” 等内容的信息时，说明安装初步完成。此时可通过手动验证命令确认是否安装成功：

uv --version

若终端显示“uv 0.7.16 (b6b7409d1 2025-06-27)” 等版本信息，则表示 uv 安装成功。

第二步：配置 wxauto-mcp

安装好 uv 后，依次在终端中执行以下命令，完成 wxauto-mcp 的克隆和同步：

git clone https://github.com/barantt/wxauto-mcp.git

cd wxauto-mcp

uv sync

第三步：Trae 与 Claude Desktop 配置
Trae 配置

在 Trae 中使用时，打开对话框设置的 “MCP 设置”，点击 “添加” 中的 “手动添加”，选择 “原始配置（JSON）”，并粘贴以下配置内容：

"mcpServers": {

  "wxauto-mcp": {

    "command": "uv",

    "args": [

      "--directory",

      "path\\to\\wxauto-mcp",

      "run",

      "wxauto_mcp.py"

    ]

  }

}

注意：其中“path\to\wxauto-mcp” 需替换为你通过git clone命令下载 wxauto-mcp 的实际路径。

Claude Desktop 配置

在 Claude Desktop 中使用时，配置方法与 Trae 类似，同样粘贴上述 JSON 配置内容即可，只需确保路径正确。

四、核心功能解析

wxauto-mcp 之所以能实现微信自动化管理，核心在于它提供的两大关键功能，满足日常沟通自动化的基本需求。

1. send_message：精准发送消息

该功能支持向指定联系人或群组发送消息，关键参数如下：

•msg：要发送的消息内容（必填）；

•to：接收消息的联系人或群组名称（必填）；

•at：在群消息中 @的人（可选）。

调用该功能后，无需手动打开微信，工具会自动完成消息发送，返回值为无。

2. get_all_messages：高效获取聊天记录

通过该功能可获取与指定联系人或群组的聊天记录，参数包括：

•who：联系人或群组名称（必填）；

•load_more：是否加载更多历史消息（可选，默认为 False）。

返回值为消息列表，仅包含类型为“friend” 的消息，每条消息都清晰记录了发送者和内容信息，为后续的记录分析、总结提供数据支持。
