# GitHub上已斩获2.7万新标，这 3 个口碑炸裂的开源项目。速速收藏！

嗨喽各位宝子！又到开源项目盘点的时间了，公众号每天都会送上3款GitHub 开源项目，供大家学习使用！

1

   

frigate

无论是应用场景还是商业价值都堪称宝藏级，GitHub上已斩获2.7万新标，它是一款开源的网络摄像头实时检测项目，跟普通摄像头不一样的是，它配备了AI检测视频画面的能力，而且一切都在本地运行。

能精准识别老人跌倒、小孩乱爬、宠物拆家、员工上了几次厕所等等，部署起来并不难。复制docker指令到命令行窗口回车执行。

下载完成后，打开它的操作界面。此时还看不到监控画面，因为没有配置视频流，所以你得有个ip摄像头。

如果没有，可以跟我一样，随便找个旧手机，给手机整个ip摄像头启动后，复制这串RTSP地址，再回到frigate的操作界面，打开配置编辑器，在path字段后粘贴RTSP地址，然后保存并重启。

此时它就开始正常工作了，你还可以在设置界面打开人脸跟踪、物体识别等，你也可以选择按照区域，看看你家二哈有没有上二楼。最绝的是，它还可以无缝集成Home assistant，让你玩转各种智能家居，解锁更多创新玩法。

https://github.com/blakeblackshear/frigate

2

   

fogsight

给他一个抽象概念或知识点，它就能生成这样的科普动画视频。它是一个AI大模型驱动的动画引擎工具，只需要输入一句话，AI直接搞定带解说配音、带双语字幕的科普动画视频。最良心的是，生成的动画可以直接导出为视频或者HTML文件，涵盖物理、数学、计算机术语和设计学等，不管是老师做课件还是博主做科普，妥妥的效率神器。
https://github.com/fogsightai/fogsight

3

   

ShareX

这款开源免费的小工具一用一个不吱声，已在github积累了夸张的超3.4万点赞，它集成了截图、录屏、GIF动图、OCR识别等十几种功能于一身。可以全屏截图、滚动截成图、窗口捕捉、录制高清视频或者动态GIF，甚至还能OCR识别图片文字，颜色拾取，屏幕标尺，生成二维码等等，可以说跟屏幕捕捉相关的功能几乎都在这了。最绝的是这么全能的工具，居然完全免费无广，这就是效率神器该有的样子。

https://github.com/ShareX/ShareX
