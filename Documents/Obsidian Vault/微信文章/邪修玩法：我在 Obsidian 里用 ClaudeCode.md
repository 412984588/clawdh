邪修玩法：我在 Obsidian 里用 ClaudeCode

起猛了!
我发现了 Obsidian 的邪修玩法，
可以直接在 Obsidian 里使用 Claude Code 了。
ㅤ
只要先安装了 Terminal 这个社区插件，
就可以直接在 Obsidian 里运行 Claude Code。
ㅤ
使用场景：
✅ 让 CLaudeCode 帮我生成每日日报
✅ 基于新创建的笔记，扮演查理芒格，给我更多思考
✅ 直接让 ClaudeCode 编辑文件，创作模板
ㅤ

如果你不知道 Claude Code 是什么❓
ㅤ
ClaudeCode 是一个在 CLI 里的 AI Agent，可以通过强大的大模型能力，在一个「环境」里帮助我们使用「工具」完成「任务」。他可以写代码，可以执行终端命令，可以访问浏览器...
ㅤ
而 Obsidian 虽然是一个知识管理软件，但本质上就是一个本地的文件夹。这个文件夹就是 Claude Code 天然的环境。
ㅤ
所以就这么一拍即合，ClaudeCode 成了 Obisidian 的最佳伙伴！
