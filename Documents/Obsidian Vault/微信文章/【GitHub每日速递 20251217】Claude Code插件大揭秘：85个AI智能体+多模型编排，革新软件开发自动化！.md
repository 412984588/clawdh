[agents] 是一个 实现智能自动化与多智能体协同 的 Python工具。简单讲，它能让多个AI代理像团队一样协作完成编程任务，特别适配Claude Code使用。适用人群：开发者、AI自动化爱好者

项目地址：https://github.com/wshobson/agents

主要语言：Python

stars: 21.9k




这个GitHub仓库 Claude Code Plugins: Orchestration and Automation 为Claude Code提供了一套全面的生产就绪系统，用于智能自动化和多智能体编排，涵盖软件开发的各个方面。

63个专注插件：粒度细、单用途的插件，优化了令牌使用并具备可组合性。
85个专业智能体：在架构、语言、基础设施、质量、数据/AI、文档、业务运营和SEO等领域拥有深厚知识的专家。
47种智能体技能：遵循渐进式披露原则的模块化知识包，提供专业知识。
15个工作流编排器：用于复杂操作（如全栈开发、安全强化、ML管道和事件响应）的多智能体协调系统。
44个开发工具：包括项目脚手架、安全扫描、测试自动化和基础设施设置等优化实用工具。
关键特性
细粒度插件架构：63个专注插件，优化了令牌使用。

100%智能体覆盖：所有插件都包含专业智能体。
智能体技能：47种专业技能，遵循渐进式披露和令牌效率原则。

工作原理

每个插件都是完全隔离的，有自己的智能体、命令和技能。用户可以按需安装插件，仅加载特定的智能体、命令和技能，避免不必要的资源加载到上下文中。同时，插件可以组合使用，以实现复杂的工作流。技能采用渐进式披露，仅在激活时加载知识。

快速开始
添加市场：使用 /plugin marketplace add wshobson/agents 命令将市场添加到Claude Code中，此时不会加载任何智能体或工具到上下文。
安装插件：使用 /plugin 命令浏览可用插件，然后使用 /plugin install 命令安装所需插件，如 python-development、javascript-typescript 等。

一键搞定！19种Linux+多版本Windows系统重装脚本大揭秘

[reinstall] 是一个 一键重装操作系统的 [脚本工具]。简单讲，它能让用户在VPS上通过一条命令快速重新安装操作系统，省去复杂流程。适用人群：需要频繁部署或重装VPS系统的运维人员和开发者。

项目地址：https://github.com/bin456789/reinstall

主要语言：Shell

stars: 9.3k

核心功能
多系统重装支持：支持一键重装到 19 种常见 Linux 发行版和 Windows 系统，涵盖了从旧版本到新版本的多种选择，如 Windows Vista 到 Windows 11、多个版本的 Debian、Ubuntu 等。还支持任意方向的重装，包括 Linux 到 Linux、Linux 到 Windows、Windows 到 Windows、Windows 到 Linux。
自动 IP 设置：能够智能设置动态或静态 IP，支持 /32、/128、网关不在子网范围内、纯 IPv6、IPv4/IPv6 在不同的网卡等复杂网络环境。
资源实时获取：不含自制包，所有资源均实时从镜像源获得，保证了资源的官方性和最新性。
驱动自动安装：在安装 Windows 时，脚本会自动按需下载安装多种公有云驱动，如 VirtIO、XEN、AWS、GCP、Azure、Intel 等相关驱动，无需手动添加。
优势
适配低配机器：专门为低配服务器（俗称“小鸡”）进行了优化，相比官方 netboot 需要更少的内存。
硬盘安全识别：全程使用分区表 ID 识别硬盘，避免写错硬盘，保障了数据安全。
引导方式多样：支持 BIOS、EFI 引导，同时也支持 ARM 服务器。
多方式查看进度：可以通过 SSH、HTTP 80 端口、商家后台 VNC、串行控制台等多种方式查看安装进度，即使安装过程出错，也能连接 SSH 手动救砖。
系统要求

原系统可以是多种类型，目标系统对内存和硬盘有不同的配置要求。例如，Alpine 系统只需要 256 MB 内存和 1 GB 硬盘，而 Windows 10、11 则需要 1 GB 内存和 25 GB 硬盘。

快速开始
下载脚本：根据当前系统是 Linux 还是 Windows，提供了不同的下载方式，同时针对国内外服务器给出了不同的下载链接。
功能使用：脚本具有多种功能，每种功能都有详细的使用说明和可选参数。
一键重装到 Linux：可以指定不同的 Linux 发行版和版本进行安装，还可以设置密码、SSH 公钥、SSH 端口等参数。
DD RAW 镜像到硬盘：支持多种格式的镜像，在 DD Windows 镜像时会自动扩展系统盘和配置 IP。
一键引导到 Alpine Live OS 内存系统：可用于备份/恢复硬盘、手动 DD、修改分区等操作。
一键引导到 netboot.xyz：可以使用商家后台 VNC 手动安装更多系统。
一键重装到 Windows：支持自动查找 ISO 或自行指定 ISO 链接，同时可以设置密码、允许被 Ping、更改 RDP 端口等参数。
注意事项
脚本不支持 OpenVZ、LXC 虚拟机，建议改用 https://github.com/LloydAsp/OsMutation。
在使用过程中，涉及到清除整个硬盘数据的操作，需要谨慎使用。
对于不同的系统和环境，有一些特定的注意事项，如安装 Windows 7 时 EFI 引导机器要开启 CSM，GCP 上安装特定 Windows ISO 可能需要特殊处理等。
其他说明
参数格式：详细说明了 -ssh-key 参数的多种填写格式。
旧版本使用：如果遇到新的 Bug，可以尝试使用旧版本，提供了使用旧版本的方法。
脚本修改：可以 Fork 本仓库，修改 reinstall.sh 和 reinstall.bat 开头的 confhome 和 confhome_cn 以及其他代码来实现自用。
应用场景
服务器系统升级或更换：当需要将服务器的操作系统从一个版本升级到另一个版本，或者从 Linux 系统更换为 Windows 系统时，可以使用该脚本进行一键重装。
测试新系统：在测试不同的 Linux 发行版或 Windows 版本时，可以快速安装和部署新系统。
数据备份和恢复：通过引导到 Alpine Live OS 内存系统，可以进行硬盘数据的备份和恢复操作。
