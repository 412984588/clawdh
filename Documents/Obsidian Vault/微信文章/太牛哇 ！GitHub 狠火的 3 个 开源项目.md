JumpServer

JumpServer 是一款开源的PAM工具，简单来说，它是一个“堡垒机”。它让你可以通过网页统一管理对服务器、数据库、容器等的访问权限。无论是 SSH、RDP 还是 Kubernetes，JumpServer 都帮你搞定！再也不用担心权限混乱、操作审计麻烦了。


https://github.com/zen-browser/desktop




2

   

Budibase

Budibase 是一个开源的 低代码 / 无代码框架，专为快速开发企业应用而设计。借助它，你能够极速构建内部工具、审批流程、数据管理系统等，无需繁杂的编程技能，甚至支持多种数据库，助你轻松把创意变成实际的应用系统。

https://github.com/Budibase/budibase

3

   

Edit

Edit 是一个由微软开源的轻量级文本编辑器，设计理念在于“简单但高效”。它保留了经典 MS-DOS 编辑器的传统之美，同时引入类似 VS Code 的现代编辑功能。你不需要复杂配置，用 Rust 打造的 Edit 性能高效、注重用户体验，很适合那些对编辑器有简单需求，又不希望陷入繁琐功能的用户。


https://github.com/microsoft/edit
