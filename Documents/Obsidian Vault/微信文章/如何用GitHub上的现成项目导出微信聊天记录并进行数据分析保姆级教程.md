# 如何用GitHub上的现成项目导出微信聊天记录并进行数据分析|保姆级教程

大家有没有这样的经历：进了非常多微信群，聊天记录动不动99+，看不过来，但是不看吧又怕错过啥。

今天就教大家，如何用AI编程解决这个难题，导出聊天记录，并一键总结群聊精华，一步步跟着我做就好：

思路：用GitHub上的chatlog项目，导出微信聊天记录，并用AI分析生成精华。

第一步，我们先下载chatlog。

登录GitHub，并在右上角搜索框搜索chatlog

搜索结果的第一个就是我们要找的项目：sjzar/chatlog，点击进去

进入页面后，往下拉，到快速入门板块，

点击“下载预编译版本”

点击“releases”

在assets里面，根据你的电脑系统选择合适的安装包，我的是Windows系统，所以选了红框框住的那个，注意不要下载倒数三行的源码等。

下载好后，我们点击exe文件安装，出现下面的页面就是安装成功了。

第二步，导出微信聊天记录。

这步需要在电脑上操作，如果你的聊天记录都在手机上，需要迁移到电脑端。

具体方法是：手机微信——我——设置——通用——聊天记录迁移与备份——迁移——迁移到电脑，这个过程可能需要十几分钟。







第三步：解密数据

我们回到Terminal UI 界面，如下图：

点击获取密钥。

如果提示无法获取，可以先关闭，点击“以管理员身份运行”

如果还是不行，可能是微信版本兼容问题，chatlog的一些版本可能还没适配最新的微信加密方式。很多用户反馈在4.0.x能正常获取 key，到了4.1.x就可能失败。那就把微信降级到 4.0.0.x 或 3.9.x。

如何查看微信版本：

电脑客户端左下角三横杠——设置——关于微信——版本信息

微信下载降级版本：

https://pc.weixin.qq.com

ok，那么问题解决后，我们重新点开chatlog，回车键获取密钥，然后再次回车键，解密数据

第四步：在终端导出聊天记录为文件

这一步我用的是本地终端+claude Code，win+R，输入cmd，回车键打开本地终端到，然后输入claude，调用claude Code。

然后用人类语言告诉她，例如：“> 用chatlog导出我在微信群聊“【深海圈】海外AI产品-交流1群(495)”2025年5月22日至10月2日的聊天记录，以纯文本形式导出。”

类似这样的表述就可以，你也可以选择其他群聊、日期，或生成csv等格式的文件；

然后需要补充什么，AI会告诉你，跟着做就行。

这一步，可能出现的问题主要有三：

1、生成的聊天记录不完整

2、无法生成文件

理论上，我们都可以直接和claude Code聊天，让她帮我们解决这些问题。

比如你不知道你要导出的群聊对应的代码是什么，可以让她帮你生成localhost的网址，让你点进去查询：

比如我遇到的是我只能导出9月23日之后的群聊消息记录，而非完整的聊天记录。

claude code检查后告诉我，是因为数据库路径不对。

后来排查原因，可能是我曾经卸载微信又没有删除历史记录，重新下载微信后，历史记录可能有两份，分别存在不同的目录下，那就要排查下，到底哪个目录下有你想要的数据。

以及我还遇到C盘空间不足，导致无法生成文件。关于C盘空间清理可以让cc教你，或者找一下专门的清理工具。

总之，最后我成功生成了深海圈全部的聊天记录，这下妈妈再也不用担心我看不过来微信群聊天啦~~
