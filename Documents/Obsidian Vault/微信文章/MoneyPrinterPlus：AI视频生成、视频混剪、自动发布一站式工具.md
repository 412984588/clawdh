# MoneyPrinterPlus：AI视频生成、视频混剪、自动发布一站式工具

> 作者: Event Stack | 发布日期: 2025年8月25日 00:22
> 原文链接: [点击查看](https://mp.weixin.qq.com/s/ihItAKcVIFeY1sPXn_IktA)

---

今天给大家推荐一款AI视频生成、视频混剪、视频合并和自动发布的开源项目—MoneyPrinterPlus，它能够让你一键批量生成短视频！

项目介绍

MoneyPrinterPlus 是一个基于AI大模型技术的智能视频生成工具，它集成了视频生成、混剪、自动发布等全套功能，让你从创意到发布全程自动化。

无论是横屏、竖屏还是方形视频，无论是知识科普、产品介绍还是情感鸡汤，它都能在几分钟内帮你生成专业级的视频内容。

核心功能：
1. AI视频批量生成

输入一个或多个关键词，系统就会自动生成视频文案、匹配素材、添加配音和字幕，最终合成短视频。

支持多种视频尺寸和布局，满足不同平台的需求。还能添加背景音乐和30多种转场特效，让视频更加专业生动。

2. 批量视频混剪

你可以准备多个视频片段和对应文案，系统会自动组合生成海量不重复的视频。

比如：你有一个关于"服装"的内容，可以准备：版型设计、面料材质、穿搭技巧等不同片段。系统会随机组合这些片段，生成多个视频，实现批量产出。

LLM模型支持： OpenAI，Moonshot，Azure，Qianfan，Baichuan，Tongyi，DeepSeek，Ollama.

语音服务支持： 本地语音、Azure、阿里云、腾讯云语音服务.

素材资源库支持： pexels、pixabay、stableDiffusion、comfyUI.

3.视频合并

添加多个视频片段，系统自动进行视频合并，生成一个或多个不同的视频。

4. 自动发布

视频生成后，支持自动发布到抖音、快手、小红书、视频号、B站等主流平台。

只需要提前登录好账号，设置好标题前缀、标签等信息，系统就能自动完成上传发布。

项目部署
前期准备
• 安装Python 3.10+
• 安装FFmpeg
拉取项目
# 拉取项目
git clone https://github.com/ddean2009/MoneyPrinterPlus.git
cd MoneyPrinterPlus
安装依赖

可以手动安装，也可用一键安装脚本.

########## 手动安装 ##########
pip install -r requirements.txt

########## 自动安装 ##########
# window 用户
setup.bat
# linux用户
setup.sh

启动项目
########## 手动启动 ##########
streamlit run gui.py

########## 自动启动 ##########
# window用户
start.bat
# linux用户
start.sh

# 浏览器访问：http://localhost:8501/
项目实测
基本配置
配置资源库

配置资源库，用于获取视频和图片素材，我使用的是Pexels（免费的），需要到对应的网站上注册一个key来实现API调用。

配置音频服务

可以使用本地语音TTS引擎，也可以用云服务，我使用的是阿里云的音频服务，也需要到对应的网站上注册获取。

配置大模型

配置大模型，用于生成视频文案，我使用的是Deepseek.

AI视频生成
1.生成视频文案和关键词

输入视频主题，点击“生成视频文案”，生成后，文案或关键词不满意可以手动改。

2.视频语音设置

设置视频的语音服务，我使用的是阿里云的音频服务（按量付费），选择云服务。配音选择了一个比较温和的女声，可以点击试听。

3.背景音乐设置

设置背景音乐所在文件夹，系统会自动读取下面的音频文件。

4.视频和字幕设置

视频可以选择布局，帧率，尺寸等基本信息，还能设置转场效果。字幕可以设置位置，字体颜色，字体大小等。

5.生成视频

点击生成视频（我用的是笔记本，没有独立显卡，大概用了2分钟左右）
视频效果看文末视频

用于心灵鸡汤类文案，素材视频是风景的视频生成效果还可以，如果生成人物类的，目前的人物都是外国人，因为使用的素材库是：Pexels

视频混剪
1.设置视频片段及文案

设置视频片段的目录（目录下可以有多个视频文件和图片），最多设置5个视频片段；设置视频文案，有多条文案则换行。系统会随机从视频片段的目录下取视频/图片/文案进行混剪。（如果文案过长，视频长度不够，系统会提示视频资源不够）

2.设置视频数量
3.生成视频

点击生成视频（混剪速度还是比较快，我设置生成1个视频，大概20秒左右生成了）
视频效果看文末视频
