# GitHub Trending 榜一的开源 Cowork 来了！4天4.5K星，首个多智能体工作流应用！

> 作者: 开源星探 | 发布日期: 2026年1月16日 18:05
> 原文链接: [点击查看](https://mp.weixin.qq.com/s/VIErF6iv0YEslN7sNaWWdA)

---

如果说前几天大家还在为 Anthropic 的 Claude Cowork 只有付费用户能用而感到遗憾，那么如今，开源社区再次证明了它的速度与能量。

就在今天，一款名为 Eigent 的项目硬生生冲上了 GitHub Trending 榜首。

在 Claude Cowork 宣布上线当天，Eigent 项目团队就紧随其后将其100%开源了出来。

它的战绩也是相当离谱：宣布彻底开源仅 4 天，就揽获 4.5K+ Star，真正的90度直线飙升呀，总收藏数已至 7.4K。

为什么这么火？因为它是 全球首个多智能体工作流桌面应用程序。 它不仅仅是一个聊天机器人，还是一个能真正接管你电脑、组建 AI 团队、并行处理任务的超级管家。

最关键的是：完全开源、支持本地模型（Ollama）、数据不出本地、还支持最新的 MCP 协议。

有了它，人人都可以做那个能控制私人电脑的“贾维斯”。

Eigent 是什么？

简单来说，Eigent 是一个桌面级、多智能体协作的通用 AI 工具。

它预定义了以下智能体工作流：

• 开发智能体：负责写代码、执行终端命令、跑脚本。
• 搜索智能体：负责联网搜索、提取网页信息。
• 文档智能体：负责整理资料、撰写文档、管理文件。
• 多模态智能体：负责看图、听录音、处理多媒体。

这套逻辑源自 CAMEL-AI 的多智能体协同理论，现在终于变成了一个人人可用的桌面 App。

从定位上看，它就是：

Claude Cowork 的开源增强版 + 多智能体工作流引擎。

核心特性

1、真正的“本地优先”

Eigent 支持接入 Ollama，这意味着你可以用本地跑的 Ollama、DeepSeek 或 Qwen 来驱动这个庞大的智能体系统。

你的文件、屏幕截图、操作记录，全部保留在本地。对于注重隐私的企业和个人开发者来说，这是刚需。

当然，如果你嫌本地慢，它也支持接入 OpenAI、Claude 等云端 API

2、多智能体并行工作流

普通的 Agent 做任务是串行的。

而 Eigent 是支持并行的。当你下令“制作一份竞品分析”时，搜索智能体在查资料的同时，文档智能体已经在准备模板，多模态智能体正在分析你上传的截图。

工作效率直接翻倍。

3、MCP 协议集成

Eigent 内置大量 MCP 工具（用于网页浏览、代码执行、Notion、Google 套件、Slack 等），并允许安装自己的工具。

为智能体配备适合您场景的工具，甚至集成内部 API 或自定义功能以增强其能力。

快速入手

Eigent 提供了三种使用模式，灵活选用。前两种个人都可使用。

1、云版本

官网：eigent.ai

官方自己上线的正式服务。可以最快体验 Eigent 多智能体，适合希望无需复杂设置即可立即使用的团队和个人。

即开即用，官方托管模型和算力，数据存云端。

2、自托管社区版

也就是自定义部署，源码安装的方式。适合偏好本地控制、数据隐私或定制的人。

具体指令：

git clone https://github.com/eigent-ai/eigent.git
cd eigent
npm install
npm run dev

还提供有更详细的前后端分离部署的指南可参考：

https://github.com/eigent-ai/eigent/blob/main/server/README_CN.md

写在最后

Eigent 它能开源即爆火，是因为它几乎踩中了当前 Agent 领域的所有关键趋势点。

它不是只给建议，而是真正执行任务：

• 管理文件
• 生成文档
• 写代码、跑命令
• 搜索资料、整理结果
• 处理图片、音频、视频

这和 Claude Cowork 的核心理念一致：让 AI 拿到“动手权”。

它不再满足于在浏览器里陪你聊天，它要进入你的文件系统，接管你的工具链，成为你的数字员工。

我只能说这个开源Cowork：本地优先 + 多智能体工作流 + 开源 + MCP，这组合太狠了！
