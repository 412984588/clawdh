# 最近 火火火 的 4 个 GitHub 开源项目

> 作者: 逛逛GitHub | 发布日期: 2026年1月15日 02:11

01

科研类 Claude Skills 

这个开源项目收集了一套针对 Claude 优化的科研类系统提示词。

把这些设定喂给 AI 后，它在处理学术任务时的表现会有质的飞跃。

无论是在解释晦涩的科学概念，还是梳理复杂的文献逻辑，甚至是在辅助撰写学术论文片段时，它都能给出更加严谨和专业的反馈。

这相当于给通用的 AI 加上了一个科研专家的滤镜：

如果你平时苦恼于如何向 AI 提问才能获得高质量的文献解读，这套指令集非常值得尝试。

它能帮助模型摒弃掉那些正确的废话，直接切入问题的核心。使用经过调优的提示词，能大大节省你反复纠正 AI 回答的时间。

开源地址：https://github.com/K-Dense-AI/claude-scientific-skills

02

DeepTutor

DeepTutor 是港大最近开源的 AI 个人学习助手，最近还挺火的。

它通过多智能体协同架构，集成了文档问答、知识可视化、练习题生成和深度研究等功能，能够基于你上传的教材或论文构建专属知识库，提供带引用的分步解答、个性化练习和真题模拟。

现在已经快 8K 的 Star 了。

这个开源项目采用苏格拉底式的教学方法，当你向它请教一个复杂的理科问题时，它不会直接告知结果。

相反，它会像一位耐心的导师那样，通过一步步的引导和反问，启发你自己去思考问题的逻辑。

这种循序渐进的交互模式能帮助学生构建起完整的解题思路。它会检测你的回答中是否存在逻辑漏洞，并给出针对性的提示。

对于学生或者自学者来说，这种引导式的交互比单纯的问答更有教育意义，能让你真正掌握知识的内核。

开源地址：https://github.com/HKUDS/DeepTutor

03

让 AI 写的 UI 变好看

这个开源项目是一套精心调优的提示词，现在已经有 1 万多的 Star 了。

目的是把 Cursor、Claude 等生成的前端 UI 更精美。

你只需要详细描述你的产品功能和目标用户，它就能从用户体验的角度给出专业的建议。

它集成了 57 种 UI 风格、95 个行业配色方案、56 种字体搭配以及多平台技术栈的最佳实践，能够根据你的需求智能推荐匹配的设计方案并生成高质量代码。

而且它会自己规划合理的界面布局，设计顺畅的交互流程，甚至是提供符合产品调性的配色方案。

除了从头设计，它还能充当评审的角色。你可以把自己现有的设计方案发给它，让它帮忙寻找逻辑漏洞或体验不佳的地方。

开源地址：https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

04

微信公众号文章下载神器

这个开源项目能把公众号文章批量下载下来。

软件运行起来后能把文章的正文、配图以及底部的精选评论全部抓取下来。

文件格式方面支持得很全面，你可以选择存成网页格式、Markdown 或者更便于打印阅读的 PDF 文件。

这些格式方便后续导入到各类笔记软件中进行二次整理。使用这个工具不需要太复杂的编程知识，配置好参数后就能自动运行。

开源地址：https://github.com/wechat-article/wechat-article-exporter
