# GitHub一巡！Claude秒变多模型指挥官，一键调度Codex+Gemini效率翻倍

项目介绍

由GuDaStudio打造的Agent Skills集合，直击Claude无法直接调用外部AI模型的痛点，通过模块化Skill机制把OpenAI Codex CLI与Google Gemini CLI无缝接入，让原型开发、代码审查、安全审计等场景瞬间获得多模型并行火力，开发者只需一句自然语言即可坐享统一Diff补丁回流，彻底告别反复Copy-Paste的低效协作。

📦 项目状态：

项目状态：积极维护中，GitHub 390⭐/22 Fork，近一月持续更新安装脚本与文档，社区活跃度高，可放心跟用。

✨ 核心功能

零侵入桥接：通过read-only沙箱把任务投递给Codex/Gemini，模型零权限写文件，代码主权牢牢握在开发者手中。

统一Diff回流：强制模型输出Unified Diff Patch，Claude一键应用并重构，保证风格一致且零冗余。

会话记忆：自动生成SESSION_ID，支持多轮上下文继续，长任务也能断点续跑。

一键安装脚本：install.sh/install.ps1支持用户级/项目级/自定义路径多策略，30秒完成环境部署。

双语文档：中英README同步维护，Workflow五阶段指引清晰，新手也能照单抓药。

🚀 快速上手 & 评价

快速上手：克隆后执行./install.sh --user --all即可把全部Skill装进Claude Code，立即体验多模型协作。

💡

综合评价：目前唯一把Codex与Gemini同时封装成Claude Skill的开源方案，无需额外API封装层，比自建桥接节省50%代码量，安全策略开箱即用，值得立刻试用。
