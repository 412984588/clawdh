X-CMD

X-CMD（读作 "X Command"）是一个小巧且功能强大的命令行工具集，能提供 100 多种针对不同应用场景的功能模块和一个包管理器，支持下载安装 500+ 个第三方开源软件工具。

在Github上已经斩获3.2k Star！

核心特性
一键启用 1000+ CLI 工具:

按需下载，像启动 App 一样启动命令行工具。

AI 加持
• AI Agent：用于不同任务的专用助手（x claude、x codex、x crush）
• 直接 AI 访问：与各种 AI 模型交互（x openai、x gemini、x deepseek）
• 任务自动化：AI 驱动的命令生成和工作流优化
环境管理
• 多语言支持：Node.js、Python、Java、Go 等
• 隔离环境：具有权限控制的沙盒执行
• 按需安装：工具在需要时才会下载并运行
• 临时会话：无需永久安装，可以直接安装试用包，
安全沙箱

提供系统权限控制沙箱和基于 Docker 的隔离沙箱 — 在安全空间中放心运行工具。

终端美化

内置终端主题与主题色切换，字体安装器，导航切换和原生的命令补全 — 美观又高效。

一致化设计 & 交互友好

统一的命令风格，直观的 TUI 交互，丝滑的流程控制，轻松的终端体验。

Hub

随时随地运行自托管脚本，即使在 Alpine / Busybox 等极简容器镜像。

极致轻量化

核心包体积 ~ 1.1 MB, 加载时间控制 ~ 100 ms.

安装使用
使用 curl 命令安装
eval "$(curl https://get.x-cmd.com)"
使用 wget 命令安装
eval "$(wget -O- https://get.x-cmd.com)"

 

 

模块

 

功能被组织为模块（mod），每一个模块都代表一个独立的功能单元，模块彼此独立，职责单一，在调用时只需记住 x <mod> 的形式，即可轻松找到并使用对应功能。




模块分为基础函数、系统命令、命令行工具、云服务Client、AI大模型、环境管理、运维、开发等，一共有333种，不得不佩服x-cmd的强大！

x top  查看进程的实时信息

x gemini init  交互模式初始化配置

x theme  浏览不同终端主题

x npm 包管理器

x java  增强型java命令
