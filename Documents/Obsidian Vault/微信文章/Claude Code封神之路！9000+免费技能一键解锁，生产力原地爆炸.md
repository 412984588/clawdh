再说原理：这系统怎么做到的？

这套系统就是个“智能星探”，24小时蹲GitHub薅羊毛。

扫货逻辑简单粗暴：专门找带SKILL.md文件的仓库。这个文件就是开发者的“技能说明书”，一看就知道这项目能不能给Claude用。

垃圾过滤系统：GitHub星标低于2星的项目直接pass。玩具项目和半成品根本进不来，省得你踩坑。

AI语义搜索：不用记专业术语，直接说“帮我处理PDF”，系统就能找到相关技能。就像跟朋友说话一样简单。

最后聊个关键的：3步搞定技能安装

直接上案例！假设你要从YouTube下载视频：

第1步：搜技能
打开skillsmp.com，搜索框输入“video-downloader”。系统秒给你推荐相关技能，星标数、简介一目了然。

第2步：看源码
点进技能详情，先看GitHub源码。高星标+最近更新的项目基本稳，垃圾项目在这关就被卡死了。

第3步：复制粘贴
将此命令复制并粘贴到 Claude Code 中以安装该技能。

复制-粘贴：/plugin marketplace add ComposioHQ/awesome-claude-skills，回车后显示marketplace安装成功。

然后执行/plugin,选择1.Browse and install plugins

继续选择awesome-claude-skills

在弹出的列表中，选择video-downloader，按下空格键选中。

输入“i”执行安装。看到“Installed 1 plugin. Restart Claude Code to load new plugins.”后必须重启CC，插件才能生效。

下次直接跟Claude Code说：

“Download this YouTube video:https://www.youtube.com/watch?v=kS1MJFZWMq4 ”

它立马给你结果（过程中会安装yt-dlp）。
