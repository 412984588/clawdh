别想太多，干就对了！完美都是干出来的。

2057 个 n8n 工作流！斩获 37.4K Star！

这是准确的开源 n8n 工作流的数字， 而且不光有工作流，还配备了一个检索系统，让你能够快速根据需求找到匹配的工作流模板。

今天二师兄着重讲的是如何搭建起来这个 n8n 工作流的检索系统。

搭起来这个系统总共需要三个步骤，包括两个前置工具的安装，git 和 python， 以及系统运行。

一 Git 安装

如果你没有本地的 git 环境，可以去下载一个

地址

https://git-scm.com/install/windows

直接下载安装即可

二 Pip 命令安装

具体可以看这个教程

https://www.runoob.com/w3cnote/python-pip-install-usage.html

三 快速检索系统部署

Github 仓库源地址：

https://github.com/Zie619/n8n-workflows

首先获取代码仓库的链接地址

新建一个文件夹，然后右键新建一个 git bash 窗口

然后在打开的窗口中执行拉取代码

git clone https://github.com/Zie619/n8n-workflows.git

代码拉取下来以后 ，需要执行以下操作

pip install -r requirements.txt

python run.py

然后就会看到提示

浏览器访问 http://127.0.0.1:8000

比如我们搜索 “gmail”，他就可以快速检索出来带有 gmail 的邮箱

直接点开就能看到几个菜单 ， 下载 json 文件，在线预览 json 文件

下载之后可以直接导入 n8n 中。

写在最后

站在巨人的肩膀上，才能走到更远。 有了这么多工作流模版，再结合检索系统，可以让你快速完成一个工作流很大一部分工作量。
