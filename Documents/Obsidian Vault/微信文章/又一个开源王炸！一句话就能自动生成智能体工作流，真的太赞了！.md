2 星辰 AstronAgent 


这样的AI智能体开发平台，其实够良心的了，没想到，我在官网上看到它们还开源了这个AstronAgent，也就是星辰Agent，如下截图所示：

这个开源的版本，也是高可用的，具备企业级可靠性，与讯飞星辰Agent平台保持一致，如下图所示：

本地部署，推荐基于Docker的方式：

# 克隆项目
git clone https://github.com/iflytek/astron-agent.git


# 进入 astronAgent 目录
cd docker/astronAgent


# 复制环境变量配置
cp .env.example .env


# 环境变量配置
vim .env

环境变量参考配置文档：

https://github.com/iflytek/astron-agent/blob/main/docs/DEPLOYMENT_GUIDE_WITH_AUTH_zh.md#%E7%AC%AC%E4%BA%8C%E6%AD%A5%E9%85%8D%E7%BD%AE-astronagent-%E7%8E%AF%E5%A2%83%E5%8F%98%E9%87%8F

启动所有服务：

docker compose -f docker-compose-with-auth.yaml up -d

启动完成后，本地访问地址：

http://localhost:8000

3 星辰 AgentRPA 

除了开源AstronAgent，他们还开源了一个同样好用的自动化工具：星辰 AgentRPA（面向流程自动化的 RPA 系统），如下图所示：

这有啥用？好处就太大了，直接把智能体的生态构建到了全网，还有物理世界中。通过此智能体，开发者比较容易就能把智能体连通到其他主流网站，如不同电商平台。

Agent+RPA，让智能体跨越应用鸿沟，打通了不同平台，再也不用手动逐个网站比价，逐个查看商品评论了， 全部交给讯飞的AgentRPA，这点真够实用。
