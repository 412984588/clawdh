一、再营销原理

再营销就是把广告投给已经和你有过接触的人群，而不是全新冷流量。通过像素（Pixel）、SDK 或 API 把用户行为收集起来，形成不同的再营销受众。

常见分层：

网站行为：访问过网站、浏览过产品页、加购但未购买、结账未完成。

社交互动：看过广告视频的用户（25%、50%、95%观看时长），点赞、评论、私信或点击过主页。

客户名单：已有客户、邮件名单、老客户再激活。

应用行为（如果有App）：安装过但未活跃、已活跃但未购买、付费但未续费。

再营销的核心目标是：

补漏（引导加购/结账的人完成购买）

升单（让已有客户复购或加购）

养熟（对冷流量有兴趣的人反复曝光，建立信任感）

二、再营销策略

你可以理解为“漏斗式再营销”：

底层漏斗（强意向）

人群：加购未购买 / 结账未完成

广告：紧迫感 + 优惠券 + 免费配送提醒

出价：高，因为转化率最高

中层漏斗（中意向）

人群：浏览过产品页 / 看过广告视频50%以上

广告：展示产品卖点、客户评价、对比图

出价：中等

上层漏斗（弱意向）

人群：和主页互动、看过广告但没进入网站

广告：品牌故事、价值观、爆款产品合集

出价：相对低，目标是建立熟悉度

客户再激活

人群：30天-180天前购买过的客户

广告：新品上架、会员折扣、限时活动

三、具体操作步骤（Meta Ads Manager）
1. 搭建追踪基础

在网站安装 Meta Pixel（或 CAPI，最好两者结合）

确保以下事件正常触发：

PageView

ViewContent（浏览产品页）

AddToCart

InitiateCheckout

Purchase

2. 创建受众

进入 受众管理器（Audiences） → 创建自定义受众（Custom Audience）：

网站行为

过去30天加入购物车但未购买的人

过去7天浏览过特定产品页但未购买的人

视频互动

看过广告视频50%以上的人

社交互动

与主页互动过（like、comment、dm）的人

客户名单

导入邮箱/手机号列表

👉 每个受众尽量覆盖 3-10万 人，太小会跑不动。

3. 结构搭建（推荐ABO+ASC组合）

建立一个 再营销Campaign（目标：Conversions 或 Sales）

广告组（Ad Set）按人群分层：

加购未买（过去14天）

浏览过产品页（过去30天）

视频观看50%以上（过去30天）

已购客户（做复购广告）

⚠️ 注意排除：比如加购未买组，要排除已购买人群，否则会重复打。

4. 广告创意方向

加购未买人群

标题：还在犹豫？库存紧张，别错过！

提供优惠码、免运费、赠品

浏览过产品人群

广告内容：展示产品细节图、用户好评视频

视频观看人群

广告内容：品牌价值 + 场景展示

复购人群

广告内容：新品推荐、老客户专享折扣

5. 出价与预算

底层漏斗（强意向）：预算 50%

中层漏斗：预算 30%

上层漏斗：预算 20%

日预算建议从 $20/组 起，根据转化情况逐步增加

6. 优化迭代

每周检查 ROAS、CTR、CPA

对低转化广告 → 替换创意

对频次过高的人群（>5） → 延长受众时间范围或换创意
