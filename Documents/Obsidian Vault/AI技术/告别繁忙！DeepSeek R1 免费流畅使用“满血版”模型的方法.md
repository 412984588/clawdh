什么是 API？怎样使用 API？

DeepSeek 和  🔗 ChatGPT 一样，除了能在官网使用外，都有提供了 API 接口可以让其他软件调用大模型，方便开发者和用户“对接”到第三方的软件里去。硅基流动 (SiliconFlow) 的服务除了有一个简单的网页版体验界面 (功能比较简单)，如下图：

但其实硅基流动的主打卖点同样还是提供 API 供用户使用的，因此我们只需要使用支持硅基流动 API 接口的客户端软件即可。

获取 API 密钥：

注册 SiliconFlow 账号并登录后，可以在页面的左下角找到 账户管理 → API 密钥，点击 新建 API 密钥，然后复制这串密钥在后面备用。
