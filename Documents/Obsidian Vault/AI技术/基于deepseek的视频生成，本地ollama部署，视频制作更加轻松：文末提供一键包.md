工具简介
只需提供一个视频 主题 或 关键词 ，就可以全自动生成视频文案、视频素材、视频字幕、视频背景音乐，然后合成一个高清的短视频。
功能特性 🎯

 完整的 MVC架构，代码 结构清晰，易于维护，支持 API 和 Web界面
 支持视频文案 AI自动生成，也可以自定义文案
 支持多种 高清视频 尺寸
 竖屏 9:16，1080x1920
 横屏 16:9，1920x1080
 支持 批量视频生成，可以一次生成多个视频，然后选择一个最满意的
 支持 视频片段时长 设置，方便调节素材切换频率
 支持 中文 和 英文 视频文案
 支持 多种语音 合成，可 实时试听 效果
 支持 字幕生成，可以调整 字体、位置、颜色、大小，同时支持字幕描边设置
 支持 背景音乐，随机或者指定音乐文件，可设置背景音乐音量
 视频素材来源 高清，而且 无版权，也可以使用自己的 本地素材
 支持 OpenAI、Moonshot、Azure、gpt4free、one-api、通义千问、Google Gemini、Ollama、 DeepSeek、 文心一言 等多种模型接入
中国用户建议使用 DeepSeek 或 Moonshot 作为大模型提供商（国内可直接访问，不需要VPN。注册就送额度，基本够用）


工具依赖
MoneyPrinterTurbo 本身是用来生成视频的，所以肯定需要：文案、配音、背景音乐、视频素材等这些基于的资源，
1、文案：根据用户输入的主题或者关键词，使用大模型deepseek等其他llm，生成文案，同时根据文案生成关键词，这个关键词用来匹配素材
2、配音：配音分为收费和免费，哈哈，相信你也不想用收费的，免费的配音是基于微软的edge-tts的
3、背景音乐：背景音乐内置了几个，可以随机使用，也可以设置使用自定义的背景音乐
4、视频素材：素材是基于，开源无版权素材网站接口获取的，主要是Pexels 或者Pixabay，我建议使用 Pixabay，这个获取key方便，免费
工具启动配置
1、首先在文末获取工具的一键包，下载后，建议先双击执行 update.bat 更新到最新代码，然后双击 start.bat 启动
这里有个bug需要解决，那就是 edge-tts的版本，在lib/python目录下执行如下命令
python.exe -m pip install edge-tts==6.1.18
不修改edge-tts的版本，程序后期使用会报错
2、启动后的界面
点击【基础配置】，进行配置。首先，如果你有llm的key，可以直接对应配置，如果没有，那么基于ollama本地模型也可以，我就基于本地ollama去讲解一下配置
     界面选择中文就行，大模型部分，目前deepseek不开放api注册了，如果你有可以选择deepseek，如果没有，可以选择本地ollama部署，我本地是16G显存，部署的14B的 deepseek-r1，配置ollama后，你也可以使用本地安装的其他模型，比如qwen2:7b等
    视频素材直接点击第二个，然后去注册，注册完后，回来重新点击就能看到appkey，然后复制到这就可以了，
工具使用
怎么使用moneyrinter生成视频呢，如下
1、写一个主题
2、点击AI主题生成
3、点击AI关键词生成
4、选择素材提供的网站，就是上边配置的Pixabay
5、选择视频样式，横屏还是竖屏
6、选择配音声音
7、选择字体
ok，可以点击生成了，是不是很简单
