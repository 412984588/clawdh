Always answer questions in Chinese （总是用中文回答问题）







你可以选择保存到当前项目，还是保存到全局。




保存成功之后，发送一条英文消息，看看返回的是不是中文。







说明你设置的 memory 就生效了。很方便的解决了不能用中文回答的困惑。




如果你想修改之前的 memory，可以通过 /memory 命令来实现。

查找历史会话

第一次使用Claude Code的时候，只能靠上下移动命令行内容来查找之前的内容。




遇到异常情况，重启Claude Code之后，就不知道怎么找回之前的会话了。




/resume命令可以新加载历史对话记录，包括完整的上下文。




在输入框中输入/resume 命令，回车就能看到历史会话记录了。







选择目标会话，回车即可快速恢复。




安装 MCP 服务

MCP定义了统一的协议规范，使AI模型能通过自然语言指令调用各类外部工具。




在 Cursor 中，安装 MCP 服务可以可视化操作，在 Claude Code 中，更简单，只需要一行命令就搞定了。




🌇

claude mcp add <name> <command> [args...]
