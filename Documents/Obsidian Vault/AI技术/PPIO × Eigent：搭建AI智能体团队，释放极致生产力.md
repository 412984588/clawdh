# 02 安装并配置Eigent

（1）进入 Eigent 官网

Eigent 是一个开源项目，你可以前往 Eigent 官网下载对应版本产品直接使用，也可以在 GitHub 获取项目进行本地部署，将数据完全储存在本地。

官网地址：https://www.eigent.ai/

GitHub 地址：https://github.com/eigent-ai/eigent










（2）配置模型

打开Eigent，点击设置→ Models → OpenAI Compatible，按以下内容配置密钥。

API Key：PPIO 官网保存的 API Key；

BASE URL：


https://api.ppinfra.com/v3/openai

模型名称：复制 PPIO 官网你所需要的模型名称







模型配置完成后，点击MCP&Tools ，选择Search(Google and Exa)，按页面提示方式获取密钥及ID，并填入对应输入框。










完成配置后，回到 Models 页面，点击验证并开启模型服务。










（3）开始任务

此时，你就可以自由设置所需 Agent，并开启你的任务。
