Claude Code 重大更新：历史会话功能终于来了！

用 Claude Code 的朋友们都知道一个“痛点”：一旦关闭窗口，会话历史就消失了。

过去，如果一个任务还没完成，我们必须手动使用 /export 命令导出文件，下次再手动加载。这个过程相当繁琐，更让人抓狂的是，偶尔忘记保存，一关机就只能拍大腿了！

说实话，“历史会话”几乎是所有AI工具的标配功能，作为最强编程智能体之一的 Claude Code 却一直缺失，确实有点说不过去。

但就在今天，这个遗憾终于被补上了！

打开 Claude Code，惊喜地发现它终于更新，现在会自动保存并展示历史会话了！
只需要在输入框使用 /resume 命令，系统就会列出你最近的会话记录。

选中你想要恢复的会话并按下回车，所有的历史上下文瞬间回归，你可以无缝衔接之前的工作。这个更新，真的太棒了！

EasyAIedu AI 创新教育项目复盘

国庆节的AI创新训练营圆满结束了。

要问孩子们具体收获了多少知识，我可能无法量化。但每当看到他们上课时专注的神情，以及下课后迟迟不愿离开、需要被“轰走”的场面，我能真切地感受到，一颗名为“创造”的火苗，已经在他们心中被点燃。

而作为组织者，我的收获更是满满：

AI 赋能课程研发：虽然我们是教育领域的新兵，但在AI的加持下，我们自研的 PBL课程助手 已经能高效生成高质量的课程方案，并成功让孩子们“沉浸”在探索的乐趣中。
把主场交还给孩子：我们彻底告别了“预制课”，将课堂的中心还给学生。我们鼓励孩子们自己规划学习路径、自主选择工具、协同解决问题。我们则退后一步，只做他们探索路上的引导者和协助者。
培养AI时代的领导力：我们引导孩子们与AI进行深度协作。利用各种强大的AI工具，让他们将自己的角色定位为项目的“导演”和“管理者”——负责组织规划、明确目标，然后将具体的执行任务交给AI。这不仅是学习知识，更是在锻炼驾驭AI的领导力。

虽然AI给这个时代带来了更多的不确定性，但是我相信我们的孩子通过这样的学习，有勇气有能力来迎接新时代的挑战！
