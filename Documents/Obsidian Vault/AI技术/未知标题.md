在用 Claude Code 写代码时，有几个命令特别实用

1. /init：让 Claude 读整个项目，生成 cloud.md，后续对话都会带上，AI 对项目理解更全面。
2. /compact：压缩对话上下文，丢掉不重要的历史内容，省 token、更专注。
3. /clear：清空对话，开启新任务前用，避免上下文干扰。
4. /think /thinkhard /thinkharder /ultrathink：控制 Claude 思考深度，任务越难就用更强的。
5. /help：查看所有命令。
6. /status：检查当前状态。
7. /reload：重新加载配置或记忆文件。
8. !：切换到命令行模式，比如 !npm run dev，结果也会写入上下文。
9. #：记忆模式，可写入项目级或用户级的长期记忆。

命令不多，但都很实用。用熟了以后，Claude Code 的体验会流畅很多。
