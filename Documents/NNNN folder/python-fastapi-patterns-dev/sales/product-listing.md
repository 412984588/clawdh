## Description
Building modern Python APIs with FastAPI means solving the same problems over and over: JWT auth, async database sessions, error handling, rate limiting, background tasks.

This pack gives you 10 complete, well-commented pattern templates you can drop straight into your project.

- **01 Basic Routing** — APIRouter, path/query params, response models, pagination

- **04 Auth JWT** — zero-dependency JWT, OAuth2 password flow, refresh tokens, role guards
- **05 Dependency Injection** — Depends(), yield sessions, class deps, lifespan

- **08 Middleware & CORS** — request ID, access logs, rate limiting, security headers
- **09 Error Handling** — custom exceptions, unified response format, global handlers
- **10 Testing Patterns** — TestClient, dependency overrides, pytest fixtures
