**Long description:**

A code asset pack for Python backend developers.

FastAPI makes it easy to *start* building APIs — but production-quality patterns for auth, database sessions, error handling, and testing are harder to find.

This pack gives you 10 templates covering the full FastAPI stack.

**Starter ($19):**
- Basic routing with APIRouter and pagination
- Pydantic v2 models with field validators
- Request validation — forms, files, cross-field rules
- JWT auth — zero-dependency, OAuth2, refresh tokens
- Dependency injection — Depends(), sessions, lifespan

**Pro ($39):**
- Background tasks with status tracking
- Async SQLAlchemy Repository pattern
- Middleware stack: rate limiting, logging, security headers
- Error handling: custom exceptions + global handlers
- Testing: TestClient, dependency overrides, pytest fixtures

All templates include README with quick-start examples.
