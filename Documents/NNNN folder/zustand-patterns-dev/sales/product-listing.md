# Zustand State Management Patterns Pack

**10 production-ready Zustand client state patterns. TypeScript. Real-world complexity.**

## Pricing

- **Starter** — $19 (5 patterns)
- **Pro** — $39 (all 10 patterns)

## What's Included

### Starter ($19)
1. **Auth Store** — User session, login/logout, token refresh, role-based selectors. devtools-enabled.
2. **Cart Store** — Shopping cart with persist (survives refresh), variant support (size/color), computed totals.
3. **UI Store** — Dynamic modal/drawer registry, toast queue with auto-dismiss, global loading overlay, command palette.
4. **Form Store** — Multi-step form with per-step validation, error tracking, progress percentage.
5. **Notification Store** — In-app notification queue with read/unread, fetch from API, action URL support.

### Pro ($39) — Everything in Starter plus:
6. **Settings Store** — Persisted user settings with `immer` for clean nested updates, server sync.
7. **Infinite List Store** — Factory function for paginated lists with filters, optimistic local delete/update.
8. **WebSocket Store** — Real-time connection state with auto-reconnect (exponential backoff), message history.
9. **Undo/Redo Store** — History tracking with `undo()`/`redo()` and `canUndo`/`canRedo` flags. Canvas example included.
10. **Slices Pattern** — Combine auth + settings + workspace into one store with partial persistence.

## Who This Is For

- React/Next.js developers who use (or are adopting) Zustand
- Teams standardizing client state management patterns
- Developers building SPAs or complex UIs (editors, dashboards, e-commerce)
