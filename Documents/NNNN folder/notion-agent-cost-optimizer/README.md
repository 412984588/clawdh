- Teams spending $500+/month on AI APIs

## Quick Start

1. Open the **Cost Dashboard** (template 01) — set your monthly budget
2. Start logging calls in the **Token Tracker** (template 02)
3. Use **Model Comparison** (template 03) to find cheaper models for simple tasks
4. Set up **Usage Alerts** (template 08) to catch spikes early
5. Generate a **Monthly Report** (template 10) for stakeholders

## File Format

All templates are **Markdown (.md)** — import into Notion, Obsidian, or any text editor.
