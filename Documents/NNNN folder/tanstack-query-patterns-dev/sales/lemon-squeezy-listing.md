# TanStack Query Patterns Pack

10 production-ready TanStack Query v5 (React Query) patterns. TypeScript. React 18+.

## Two Tiers

**Starter — $19**
- Basic Query (query key factory, staleTime, retry)
- Pagination (keepPreviousData, flicker-free)
- Infinite Scroll (cursor-based, IntersectionObserver)
- Mutations (create/update/delete, cache invalidation)
- Prefetching (Next.js App Router SSR + hover prefetch)

**Pro — $39**
All 5 Starter patterns plus:
- Real-Time Polling (auto-stop condition)
- Dependent Queries (sequential + fan-out)
- Parallel Queries (useQueries + combine)
- Cache Invalidation (key hierarchy, staleTime vs gcTime)
- Optimistic Updates (snapshot + rollback)

## Requirements
- @tanstack/react-query v5
- React 18+
