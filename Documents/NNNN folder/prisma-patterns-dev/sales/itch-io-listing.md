# Prisma Patterns Dev Pack — itch.io Listing

**Title:** Prisma Patterns Dev Pack

**Short description:** 10 TypeScript templates for production Prisma ORM patterns.

**Classification:** Tools / Developer Tools

**Description:**

A collection of TypeScript implementations covering the most important Prisma ORM patterns for backend development.

Each template is a standalone file with exported functions you can drop into your project.

**Included templates:**
1. Basic CRUD — create, findUnique/Many, update, upsert, delete, aggregate
2. Relations — one-to-many, many-to-many, nested writes, include, select
3. Migrations — dev/deploy workflow, seeding, validation, column rename
4. Filtering & sorting — operators, relation filters, dynamic where builder
5. Transactions — sequential, interactive, retry, nested writes
6. Soft delete — deletedAt Client Extension, restore, purge (Pro)
7. Pagination — offset, cursor, Relay connection, keyset (Pro)
8. Raw queries — $queryRaw, $executeRaw, JSONB, parameterized SQL (Pro)
9. Middleware & extensions — computed fields, methods, query hooks (Pro)
10. Testing — test client, transaction rollback, fixtures, mocks (Pro)
