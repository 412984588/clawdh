**Title:** Fastify Patterns Dev Pack

**Short description:** 10 TypeScript templates for production Fastify HTTP server patterns.

**Classification:** Tools / Developer Tools

**Description:**

A collection of TypeScript implementations covering the most important Fastify patterns for backend development.

**Included templates:**
1. Basic routes — GET/POST/PUT/DELETE, params, query strings, prefixing
2. Schema validation — TypeBox, JSON Schema, response serialization
3. Plugins — fp() pattern, decorators, encapsulation
4. Hooks & lifecycle — onRequest/preHandler/onSend, content type parser
5. Authentication — JWT, API keys, role-based access
6. Error handling — AppError classes, global error handler (Pro)
7. File upload — multipart, validation, streaming (Pro)
8. WebSockets — rooms, broadcast, typed messages (Pro)
9. Testing — inject(), app factory, mock dependencies (Pro)
10. Deployment — graceful shutdown, health checks, CORS (Pro)
