# Changelog

All notable changes to Fastify Patterns Dev Pack are documented here.

## [1.0.0] - 2024-01-01

### Added
- `01-basic-routes` — GET/POST/PUT/DELETE, params, query strings, reply helpers, route prefixing
- `02-schema-validation` — TypeBox schemas, JSON Schema, body/params/response validation
- `03-plugins` — fp() plugin pattern, decorators, encapsulation, module augmentation
- `04-hooks-lifecycle` — onRequest, preHandler, onSend, onResponse, onClose, content type parser
- `05-authentication` — @fastify/jwt, API key auth, role-based access, refresh tokens
- `06-error-handling` — Custom AppError hierarchy, setErrorHandler, 404 handler
- `07-file-upload` — @fastify/multipart, single/multiple files, form fields, streaming
- `08-websockets` — @fastify/websocket, rooms, broadcast, typed messages, heartbeat
- `09-testing` — inject() pattern, app factory, makeRequest helper, mock dependencies
- `10-deployment` — Graceful shutdown, health checks, helmet, CORS, compression, rate limiting
- Starter ZIP (templates 01–05, $19)
- Pro ZIP (all 10 templates, $39)
