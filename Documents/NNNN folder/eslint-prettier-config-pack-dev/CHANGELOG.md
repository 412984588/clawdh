# Changelog

## v1.0.0 — 2026-03-21

### Added
- 10 ESLint + Prettier config templates
- 01-react-typescript: flat ESLint config with react-hooks and react-refresh
- 02-nextjs: Next.js official presets + Tailwind CSS class sorting
- 03-nodejs-api: strict TypeScript API config with security plugin
- 04-typescript-strict: maximum strictness with type-checked rules
- 05-vue: Vue 3 Composition API + TypeScript
- 06-svelte: SvelteKit + TypeScript support
- 07-monorepo: root-level config for Turborepo/pnpm workspaces
- 08-library: publishable npm library config with explicit return types
- 09-vitest: test-specific rules with eslint-plugin-vitest
- 10-legacy-js: .eslintrc.json for plain JS ES2021+ projects
- Starter tier ($19, templates 01–04)
- Pro tier ($39, all 10 templates)
