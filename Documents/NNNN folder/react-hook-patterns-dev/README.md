# React Hook Patterns

**10 production-ready custom React hooks. Copy, drop in, ship.**

Stop rewriting the same hooks every project. Every common pattern — debounce, localStorage, media queries, intersection observer, fetch, clipboard, keyboard shortcuts, and more — typed, tested, and ready to use.

## What's Included

| Hook | Description | Tier |
|------|-------------|------|
| `useDebounce` | Debounce any value with configurable delay | Starter |
| `useLocalStorage` | SSR-safe localStorage with JSON serialization | Starter |
| `useMediaQuery` | Responsive breakpoint detection | Starter |
| `useIntersectionObserver` | Lazy loading, infinite scroll triggers | Starter |
| `useFetch` | Simple data fetching with loading/error/data states | Starter |
| `usePrevious` | Track the previous value of any state | Pro |
| `useClipboard` | Copy to clipboard with status feedback | Pro |
| `useEventListener` | Typed event listeners with auto-cleanup | Pro |
| `useKeyPress` | Keyboard shortcut detection | Pro |
| `useOutsideClick` | Detect clicks outside a ref element | Pro |

## Tiers

- **Starter** ($19): Hooks 01–05 — the most commonly needed hooks
- **Pro** ($39): All 10 hooks — complete custom hook library

## Usage

Copy `hook.ts` into your project's `hooks/` directory and import:

```ts
import { useDebounce } from './hooks/useDebounce'
```

## Requirements

- React 16.8+ (hooks support)
- TypeScript 4.9+
