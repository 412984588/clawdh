# Personal Goal Tracker — Notion Template

A structured system to set, track, and achieve goals across every life area.
Built on the OKR (Objectives & Key Results) framework, simplified for personal use.

---

## Database Schema: Goals

> Create a **Database** in Notion with these properties:

| Property Name | Type | Options / Notes |
|--------------|------|----------------|
| Goal | Title | — |
| Life Area | Select | 🏋️ Health / 💰 Finance / 💼 Career / 🎓 Learning / ❤️ Relationships / 🎨 Creative / 🧘 Wellbeing / 🏠 Home |
| Status | Select | 💭 Idea / 🎯 Active / ✅ Achieved / 🚫 Abandoned |
| Timeframe | Select | This Week / This Month / Q1 / Q2 / Q3 / Q4 / This Year / 3 Years |
| Progress | Number | % format |
| Target Date | Date | — |
| Why (Motivation) | Text | Your deeper reason |
| Key Results | Text | 2–3 measurable outcomes |
| Review Date | Date | When to next check in |
| Parent Goal | Relation | Link to bigger goal (for multi-level) |

## Goal-Setting Rules (Built Into This Template)

| Rule | Why |
|------|-----|
| Maximum 3 active major goals | Focus beats breadth |
| Each goal has 2–3 Key Results | If you can't measure it, you can't track it |
| Goals have a specific date | "Someday" = never |
| Goals connect to a life area | Ensures balance |
| Review every 4 weeks | Goals drift without check-ins |

---

## Goal Grading Scale

At the end of the period, grade each Key Result:

| Grade | Score | Meaning |
|-------|-------|---------|
| ✅ Achieved | 1.0 | Fully completed |
| 🟢 Strong | 0.7–0.9 | Mostly achieved |
| 🟡 Partial | 0.4–0.6 | Significant progress |
| 🔴 Missed | 0–0.3 | Little progress |

**Target score: 0.7** — if you're scoring 1.0 every time, your goals are too easy.
