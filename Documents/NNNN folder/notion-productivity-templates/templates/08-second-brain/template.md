# Second Brain — Notion Knowledge Management Template

Build a personal knowledge system using the PARA method (Projects, Areas, Resources, Archive).
Based on Tiago Forte's "Building a Second Brain."

---

## The PARA System

| Level | What Goes Here | Example |
|-------|---------------|---------|
| **P**rojects | Short-term efforts with a deadline | "Launch new website by Oct 1" |
| **A**reas | Ongoing responsibilities without deadlines | "Health", "Finances", "Career" |
| **R**esources | Topics you want to reference later | "Marketing strategies", "Book notes" |
| **A**rchive | Inactive items from above 3 | Old projects, past notes |

**Rule:** Every note lives in exactly one place.

## Progressive Summarization Method

When you capture a note, go through these levels over time:

| Level | When | What you do |
|-------|------|-------------|
| Layer 0 | Immediately | Save the raw content |
| Layer 1 | On review | Bold the most important passages |
| Layer 2 | On second review | Highlight the most important bolded passages |
| Layer 3 | When using the note | Write an executive summary at top |

*Only go to Layer 2+ for notes you return to more than once.*

## Capture Habits

Build a capture practice — the "inbox" for your second brain:

### What to Capture
- Quotes that resonate
- Ideas that come to you during walks / showers
- Things that surprise you
- Anything you might want to reference later
- Problems you want to solve

### What NOT to Capture
- Everything (curation is key)
- Things you can Google in 5 seconds
- Information you'll never act on

### Capture Tools → Notion Inbox

| Source | Capture Method |
|--------|---------------|
| Books | Kindle highlights → import or manual copy |
| Articles | Web clipper (Notion Web Clipper extension) |
| Podcasts | Voice note → transcribe → Notion |
| Conversations | Quick note in Notion mobile app |
| Thoughts | Notion inbox or daily note |

## Search Strategy

When you can't find something:

1. **Search by title** — exact keywords from what you remember
2. **Search by tag** — browse by #tags
3. **Search by area** — check the most relevant Area page
4. **Search by time** — "what was I working on in [month]?"
5. **Full-text search** — use Notion's search (Cmd/Ctrl + P)
