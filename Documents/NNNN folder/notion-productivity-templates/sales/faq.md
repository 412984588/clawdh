# FAQ — Notion Productivity Templates Pack

## Do I need a paid Notion plan?

No. All templates work on Notion's free plan. The database features used (properties, views, filters) are available on free accounts. The only exception: if you want to share databases with teammates, you'll need a paid plan.

## How do I import the templates into Notion?

1. Open the `.md` file in any text editor
2. Copy the content
3. Create a new Notion page
4. Paste — Notion automatically formats Markdown (headings, tables, checklists)
5. For database sections, use Notion's `/database` command and add the properties listed in the schema table

## Do I need Notion to use these?

No — the Markdown files work in any app. Obsidian users can import them directly. Google Docs, VS Code, Typora, iA Writer, Bear, and any text editor all handle Markdown.

## Are these actual Notion templates (shareable links)?

No — they're Markdown files you customize and own. The advantage: you control everything, no external dependencies, and they work in any Markdown-based app. The trade-off: you spend 5–10 minutes setting up each template.

If you want a click-to-duplicate Notion link, each template's setup instructions tell you exactly how to build it — it's straightforward.

## What's the difference between Starter and Pro?

**Starter ($29)** is the core system: manage projects, track goals, build habits, and do weekly + monthly reviews. These 5 templates work together as a complete productivity system.

**Pro ($49)** adds the knowledge management layer: capture what you read and learn (06–08), manage relationships (09), and plan each day at a granular level (10). Get Pro if you're serious about building a long-term knowledge system.

## How long does setup take?

Each template takes 5–10 minutes to paste in and customize. The whole Pro pack takes about 1–2 hours to set up fully. Most people set up one template at a time as they need it.

## Will these work for students?

Yes. Templates 01 (project management), 03 (habit tracker), 04 (weekly review), 06 (reading notes), and 10 (daily planner) are especially useful for students.
