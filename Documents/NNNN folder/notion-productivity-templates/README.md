# Notion Productivity Templates Pack

10 beautifully structured Notion templates to run your projects, track goals, build habits, and manage your entire life from one workspace.

## What's Inside

| # | Template | What It Does |
|---|----------|--------------|
| 01 | Project Management | Kanban board, task database schema, timeline views, team assignments |
| 02 | Goal Tracker | OKR-based annual/quarterly goal system with monthly check-ins |
| 03 | Habit Tracker | Daily log, monthly grid, habit design worksheet, streak milestones |
| 04 | Weekly Review | 20-minute Sunday ritual — close the week, plan the next |
| 05 | Monthly Review | Deep 45-minute life audit with life area scoring and goal updates |
| 06 | Reading Notes | Book library, per-book notes with progressive summarization |
| 07 | Meeting Notes | Universal, 1:1, and client call templates with action item tracking |
| 08 | Second Brain | PARA system (Projects/Areas/Resources/Archive) knowledge management |
| 09 | Personal CRM | Contact profiles, follow-up tracker, networking notes, birthday reminders |
| 10 | Daily Planner | MIT method, time blocking, evening reflection, shutdown ritual |

## Tiers

- **Starter** ($29) — Templates 01–05: Projects, goals, habits, weekly & monthly reviews
- **Pro** ($49) — All 10 templates: Full productivity operating system

## How to Use

Each template is a Markdown (`.md`) file. Import into Notion by:
1. Creating a new Notion page
2. Pasting the Markdown content (Notion imports it automatically)
3. Converting tables to Notion databases where indicated

## Format

All files are `.md` — compatible with Notion, Obsidian, Typora, and any text editor.
