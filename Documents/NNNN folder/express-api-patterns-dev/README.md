# Express API Patterns

**10 production-ready Express.js TypeScript REST API patterns. Stop reinventing the wheel.**

Every pattern you need to build a robust Express API — routing, error handling, validation, JWT auth, rate limiting, file upload, logging, CORS/security, database integration, and API versioning.

| 01-basic-routing | Router, params, query, response helpers | Starter |
| 02-error-handling | Custom errors, global handler, async wrapper | Starter |
| 03-validation-middleware | Zod validation, request schema, error formatting | Starter |
| 04-auth-jwt | JWT middleware, login/register, refresh tokens | Starter |
| 05-rate-limiting | Per-route limits, custom key generators, Redis store | Starter |
| 06-file-upload | Multer, file validation, S3-compatible storage | Pro |

| 10-api-versioning | URL versioning, header versioning, deprecation | Pro |
