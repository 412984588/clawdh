# FAQ — Express API Patterns

**Q: Which Express version is supported?**
Express 4 and Express 5. The patterns use the standard middleware API that works in both versions.

**Q: Do I need TypeScript?**
Yes. All templates are written in TypeScript and use typed request/response patterns.

**Q: Can I use these in commercial projects?**
Yes. The license allows unlimited personal and commercial use with a one-time purchase.

**Q: Are external dependencies required?**
Template 03 uses Zod for validation. Template 04 uses only Node.js built-ins (crypto). Most other templates use only Express and Node.js built-ins. Dependencies are clearly documented in each template.

**Q: What's the difference between Starter and Pro?**
Starter (01–05) covers the core patterns you need in every API. Pro (01–10) adds advanced patterns: file upload, structured logging, CORS/security headers, database integration, and API versioning.

**Q: Will I get updates?**
Yes. Bug fixes and compatibility updates are included at no extra cost.
