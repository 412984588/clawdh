# Express API Patterns

**10 production-ready Express.js TypeScript patterns. The boilerplate you need to stop copying from old projects.**

**Starter ($19) — Templates 01–05:**
- Basic Routing — typed requests, response helpers, CRUD, pagination
- Error Handling — custom error classes, asyncHandler, global error middleware
- Validation Middleware — Zod schemas for body/query/params, error formatting
- JWT Auth — sign/verify, authenticate, authorize, refresh token rotation
- Rate Limiting — fixed window, sliding window, per-route, per-user

**Pro ($39) — All 10 templates:**
- Everything in Starter, plus:
- File Upload — multer, MIME validation, magic bytes, S3-compatible storage
- Request Logging — structured JSON logs, request ID, correlation, slow alerts
- CORS & Security — CORS middleware, security headers, CSP, CSRF protection
- Database Patterns — Repository, Service layer, transactions, query builder
- API Versioning — URL + header versioning, deprecation notices, multi-version routing

## Requirements

- Node.js 18+
- TypeScript 5+
- Express 4 or 5
