# Vitest Test Patterns

**10 production-ready Vitest test templates. Copy, adapt, ship.**

Stop writing boilerplate tests. Every pattern you need — unit functions, async, mocking, components, snapshots, parameterized — ready to drop into any TypeScript/React project.

| Template | Description | Tier |
|----------|-------------|------|

| 02-class-testing | OOP class testing with beforeEach/afterEach | Starter |
| 03-async-promises | async/await, promise rejection, race conditions | Starter |
| 04-timer-mocking | vi.useFakeTimers, advance timers, debounce | Starter |
| 05-module-mocking | vi.mock, vi.spyOn, vi.fn patterns | Starter |
| 06-component-testing | React Testing Library + Vitest | Pro |

| 08-snapshot-testing | Inline snapshots, component snapshots | Pro |
| 09-error-handling | toThrow, rejection, error boundaries | Pro |
| 10-parameterized-tests | test.each, describe.each, data-driven | Pro |

## Tiers

- **Starter** ($19): Templates 01–05 — covers 90% of unit testing needs
- **Pro** ($39): All 10 templates — complete testing toolkit

## Usage

Each template is self-contained:
1. Copy `example.test.ts` into your project
2. Replace placeholder types/functions with your own
3. Run `npx vitest run`

## Requirements

- Node.js 18+
- `vitest` (any recent version)
- For template 06: `@testing-library/react`, `@testing-library/user-event`
