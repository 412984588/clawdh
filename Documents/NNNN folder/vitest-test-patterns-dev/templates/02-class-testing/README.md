# 02 — Class Testing

Test OOP classes with `beforeEach` instance reset, `afterEach` cleanup, and nested `describe` for logical grouping.

## Patterns shown

- `beforeEach` for fresh state isolation
- `afterEach` for cleanup
- Nested `describe` blocks per method
- Exception assertions on invalid state

## Run

```bash
npx vitest run example.test.ts
```
