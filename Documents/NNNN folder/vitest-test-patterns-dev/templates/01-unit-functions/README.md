# 01 — Unit Functions

Test pure functions with Vitest matchers. Covers `toBe`, `toEqual`, `toBeCloseTo`, `toThrow`, and describe/it structure.

## Patterns shown

- Numeric equality (`toBe`, `toBeCloseTo`)
- Exception assertions (`toThrow` with message)
- Range/boundary testing (`clamp`)
- Array equality (`toEqual`)

## Run

```bash
npx vitest run example.test.ts
```
