## Product Name
Social Media Content Calendar Pack

## Short Description (160 chars)
30-day content calendars + caption formulas + analytics tracker for 7 platforms. Stop guessing, start growing.

## Long Description

Growing on social media isn't hard because the algorithms are unfair. It's hard because most creators don't have a system.

This pack gives you the system.

**5 platform-specific 30-day content calendars:**

- **Instagram** — daily post plan (format + content type + caption hook for all 28 days), caption formulas (AIDA, story, value drop), Reel hook library, Stories 5-slide framework
- **LinkedIn** — 3 posts/week plan, contrarian / story arc / list post formulas, profile optimization checklist, engagement tactic guide
- **Twitter/X** — daily tweet ideas + weekly thread topics, 9-tweet thread template, 5 tweet formulas, reply strategy for organic growth
- **TikTok** — daily video ideas, hook formula library, algorithm signals ranked by impact, weekly TikTok rhythm
- **Facebook** — daily post plan, Groups strategy, question post formulas, Facebook Live guide, ad warm-up checklist

### Pro Pack — $39

Everything in Starter, plus:

- **YouTube** — weekly video plan, tutorial/listicle/opinion video structures, CTR-optimized title formulas, YouTube SEO checklist
- **Pinterest** — 30-day task plan, pin design specs, high-performing pin layouts, keyword placement guide
- **Content Batching System** — 8-hour Content Day schedule, 50 evergreen idea prompts, repurposing matrix (one piece → 5 platforms)
- **Analytics Tracker** — per-platform weekly/monthly tables, engagement rate formulas, cross-platform traffic attribution, quarterly goals
- **90-Day Strategy** — audience definition worksheet, platform selection guide, Month 1/2/3 priorities, offer launch calendar, quarterly retrospective
