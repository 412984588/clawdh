# Pinterest Content Calendar

Pinterest is a search engine, not a social network. Build a library of evergreen pins that drive traffic for years.

## Platform Overview

| Metric | Target | Notes |
|--------|--------|-------|
| Best posting frequency | 5–15 pins/day | Pinterest rewards daily volume (schedule with Tailwind) |
| Pin lifespan | Months to years | Unlike Instagram — pins compound over time |
| Best content formats | Vertical images (2:3), idea pins, video pins | 2:3 ratio (1000×1500px) dominates feed |
| Traffic destination | Blog, shop, lead magnet, product | Every pin = potential traffic forever |
| Best niches | Food, home decor, fashion, DIY, health, finance, travel | But works for any visual niche |

---

## Content Mix

| Type | Frequency | Goal |
|------|-----------|------|
| Blog/article pins | 5–7/day | Website traffic |
| Product pins | 2–3/day | Sales (Shopify, Etsy) |
| Infographic pins | 1–2/day | Saves and shares |
| Idea pins (native) | 1–2/week | Followers and engagement |
| Seasonal/trend pins | Weekly | Timely traffic spikes |
| Repins (curated) | 2–5/day | Fill posting schedule, signal niche relevance |

## Pin Image Design Guide

### Specs and Format
| Element | Specification |
|---------|--------------|
| Aspect ratio | 2:3 (1000×1500px recommended) |
| File type | JPG or PNG (under 10 MB) |
| Text on image | Large, readable at mobile size |
| Brand colors | Consistent palette across all pins |
| Logo | Small watermark in corner |

### High-Performing Pin Layouts

**The List Pin:**
```
[Headline: "10 Ways to [Result]"]
[Image or flat-lay background]
[Numbered list preview (first 3 items)]
[Your URL or logo]
```

**The Tutorial Pin:**
```
[Hook: "How to [Result] in [Timeframe]"]
[Process visual or before/after]
[Step numbers: 1, 2, 3...]
[Your brand name]
```

**The Quote/Inspiration Pin:**
```
[Clean background]
[Large quote text]
[Attribution + your handle]
```

**The Infographic Pin:**
```
[Title at top]
[Icon + data for each point]
[Source + URL at bottom]
[Tall vertical layout (1:2.1 ratio)]
```

## Pinterest SEO: Keywords Where They Count

Add keywords in priority order:

1. **Pin title** — most important, front-load keywords
2. **Pin description** — 200–500 words, naturally written, multiple keyword variations
3. **Board title** — use exact search terms people type
4. **Board description** — 2–3 sentences with keywords
5. **Profile description** — niche + who you help + keywords

**Keyword research tools:**
- Pinterest search bar autocomplete (free, best source)
- Pinterest Trends (trends.pinterest.com — free)
- Ubersuggest, Ahrefs (for cross-referencing)
