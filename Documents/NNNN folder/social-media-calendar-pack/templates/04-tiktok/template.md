# TikTok Content Calendar

Grow on TikTok with trending audio, native formats, and a consistent posting rhythm.

## Platform Overview

| Metric | Target | Notes |
|--------|--------|-------|
| Best posting frequency | 1–4x/day | TikTok rewards volume, but quality matters |
| Peak posting times | 7–9am, 12–3pm, 7–11pm | Test daily — TikTok distributes globally |
| Best content formats | Tutorials, POV, storytelling, duets, trends | Short-form video only |
| Optimal video length | 15–60 sec (sweet spot), up to 10 min | Under 30s gets boosted on initial push |
| Hook window | First 1–3 seconds | If they don't stop scrolling, nothing else matters |

## Hook Formula Library

TikTok hooks must stop the scroll in 1–3 seconds. Use these structures:

### The Direct Question
```
"[Question that your audience is already asking themselves]?"
```

### The Surprising Statement
```
"[Thing everyone believes] is actually [opposite]."
```

### The POV Setup
```
"POV: [Relatable scenario in your niche that triggers curiosity]"
```

### The List Tease
```
"[Number] [things/mistakes/secrets] about [topic] — #[last number] changed everything."
```

### The Before State
```
"If you're struggling with [problem], watch this whole video."
```

### The Achievement Tease
```
"I went from [bad state] to [good state] in [timeframe]. Here's how."
```

## TikTok Captions and Hashtags

**Caption structure (150 chars max):**
```
[1 line reinforcing the hook]
[Optional CTA — "Follow for more [topic]"]
[2–3 hashtags]
```

**Hashtag strategy:**
- 1 broad niche hashtag (#[yourniche])
- 1 specific topic hashtag (#[subtopic])
- 1 content type tag (#tutorial, #storytime, #pov)
- Avoid #fyp, #foryou — they're too saturated to drive placement

## TikTok Algorithm Signals (Ranked by Impact)

| Signal | Weight | How to optimize |
|--------|--------|-----------------|
| Completion rate | Very high | Hook strong, no dead air, end with cliffhanger |
| Rewatches | High | Dense value, "watch again" moment |
| Shares | High | Create content people send to someone specific |
| Comments | Medium | Ask a direct question at end |
| Likes | Medium | Baseline positive signal |
| Follows | Medium | Appear consistently so they recognize you |
