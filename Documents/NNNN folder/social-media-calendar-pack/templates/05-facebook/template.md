# Facebook Content Calendar

Build a loyal Facebook community with a mix of posts, groups, and video content.

## Platform Overview

| Metric | Target | Notes |
|--------|--------|-------|
| Best posting frequency | 1–2x/day | Consistency beats volume on Facebook |
| Peak posting times | Wed–Fri, 9am–1pm | Midweek mornings perform best |
| Best content formats | Video, link posts, text updates, events | Video gets 3× more reach than other types |
| Optimal post length | 40–80 chars for text, 1–3 min for video | Shorter text = more engagement on feed |
| Best features | Groups, Live, Events, Reels | Groups have 10× higher organic reach than Pages |

| Type | Frequency | Goal |

## Facebook Post Formulas

### The Share-Worthy Insight
```
[Surprising or counterintuitive fact]

Most people don't realize that [explanation].

Here's what to do instead: [actionable advice]

[Question to drive comments]
```

### The Story Post
```
[Year or timeframe] ago, I [where you were].

[The problem or struggle]

Everything changed when I [turning point].

Now [where you are].

The lesson: [one-line takeaway]

Has [similar situation] ever happened to you?
```

### The Value List
```
[Number] things that [changed/improved] my [area]:

1. [Point]
2. [Point]
3. [Point]
4. [Point]
5. [Point]

Which one surprised you most?
```

### The Question Post (Highest engagement format)
```
[Thought-provoking or relatable question about your niche]?

I'll go first: [your honest answer]
```

## Facebook Groups Strategy

Groups outperform Pages 10:1 for organic reach. Build one:

**Group setup:**
- Name: "[Your Niche] Community" or "[Your Name]'s [Topic] Hub"
- Type: Private (visible) — creates exclusivity without blocking discovery
- Rules: 3–5 clear rules (no self-promotion spam, respect, value-add only)

**Group content rhythm:**
| Day | Post Type |
|-----|-----------|
| Mon | Weekly challenge or prompt |
| Wed | Educational post or resource |
| Fri | Win of the week / member spotlight |
| Weekend | Open discussion thread |
