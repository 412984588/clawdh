## Product Description

### The five Git problems developers never solve

Every developer knows their Git workflow has rough edges. Most never fix them because writing a pre-commit hook from scratch takes an afternoon, configuring delta requires reading three blog posts, and nobody can remember the gitattributes syntax for binary files.

These are solved problems. The solutions just aren't packaged together.
