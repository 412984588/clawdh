# Vue 3 Composition API Patterns

10 production-ready Vue 3 patterns covering `<script setup>`, composables, Pinia, Vue Router, reactive forms, provide/inject, async patterns, TypeScript, transitions, and testing.

## What's Inside

| # | Template | Description |
|---|----------|-------------|
| 01 | Script Setup | defineProps, defineEmits, defineExpose, useAttrs, v-model |
| 02 | Composables | useFetch, useToggle, useDebounce, useLocalStorage, useAsync |
| 03 | Pinia Store | Option store, setup store, cross-store composition, plugins |
| 04 | Vue Router | Guards, nested routes, lazy loading, typed params, scroll |
| 05 | Reactive Forms | Field validators, cross-field validation, submit state |
| 06 | Provide / Inject | Typed injection keys, Theme/User/Toast providers, plugins |
| 07 | Async Patterns | Suspense, defineAsyncComponent, race cancellation, optimistic UI |
| 08 | TypeScript + Vue | Generic components, typed emits, generic composables |
| 09 | Transitions | CSS classes, JS hooks, TransitionGroup, GSAP integration |
| 10 | Testing Patterns | Vitest, @vue/test-utils, composable testing, dependency mocking |

## Tiers

- **Starter** ($19) — Templates 01–05: core Composition API patterns
- **Pro** ($39) — All 10 templates including advanced async, TS, and testing

## Quick Start

```bash
npm create vue@latest my-app
cd my-app && npm install
# Copy patterns from templates/ into your components
```

- Vue 3.4+
- TypeScript 5.0+
