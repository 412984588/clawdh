**Short description:**
10 production-ready Vue 3 Composition API templates. Composables, Pinia, Router, forms, TypeScript, and testing — copy-paste ready.

**Long description:**

A code asset pack for Vue 3 frontend developers.

Vue 3's Composition API gives you a lot of flexibility — but that also means there are many ways to do things wrong. This pack shows you the patterns that work in production.

**Starter ($19):**
- script setup with typed props, emits, and v-model
- Reusable composables (useFetch, useToggle, useDebounce)
- Pinia setup store with cross-store composition
- Vue Router 4 with auth guards and lazy loading
- Reactive form validation without third-party libraries

**Pro ($39):**
- Provide/Inject with TypeScript injection keys
- Async patterns: Suspense, race condition handling, optimistic UI
- Generic TypeScript components and composables
- Transition and animation patterns (CSS, JS hooks, GSAP)
- Testing with Vitest and @vue/test-utils

All templates include TypeScript implementation and README.
