## DIY Signage

| Sign | Materials | Template |
|------|----------|---------|
| Welcome sign | Foam board or acrylic + vinyl letters or hand-lettered | "Welcome to the wedding of [Bride] & [Groom]" |
| Seating chart | Mirror, window frame, or poster board | Guest names by table number |
| Bar menu | Chalkboard or framed print | "Tonight's Menu: [Signature cocktails]" |
| Cards & gifts | Small frame + printed card | "Cards & Gifts — Thank You!" |
| Hashtag sign | Small frame | "Share the love! #[YourHashtag]" |
| Dessert table | Small sign or tent cards | "Love is sweet — take a treat!" |
| Photo booth | Frame + easel | "Grab a prop, strike a pose!" |

## DIY Timeline (Work Backwards from Wedding Date)

| When | What to Do |
|------|-----------|
| 3–4 months before | Plan all DIY projects, order materials |
| 2–3 months before | Test build 1 of each project (discover problems early) |
| 1–2 months before | Start assembly on items that won't spoil (signs, table numbers, favors) |
| 2 weeks before | Finish all non-floral DIY projects |
| 2–3 days before | Assemble flower arrangements (if doing fresh flowers) |
| Day before | Pack everything in labeled boxes for venue transport |
| Wedding morning | Set up crew assembles at venue using the setup guide |
