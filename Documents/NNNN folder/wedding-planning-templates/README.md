# Wedding Planning Templates

A practical wedding planning system for couples who want structure without
buying complicated software. The pack covers the full planning arc: checklist,
budget, vendors, guests, ceremony timeline, seating, registry, honeymoon, and
DIY decor.

## What's Included

### Starter Pack ($19) — Templates 01–05
| Template | What You Get |
|----------|-------------|
| 01-master-checklist | 12-month wedding planning checklist with ceremony and reception milestones |
| 02-budget-tracker | Budget worksheet for venue, dress, flowers, cake, photography, and guest-count scenarios |
| 03-vendor-manager | Comparison sheet for venue, florist, DJ, caterer, baker, and photo vendor outreach |
| 04-guest-list | RSVP tracker with meal choices, plus-ones, gift notes, and seating flags |
| 05-timeline-day-of | Ceremony-to-reception timeline, vendor arrival grid, and emergency contacts |

## Format

Markdown files — open in Notion, Obsidian, Google Docs, or any text editor.
