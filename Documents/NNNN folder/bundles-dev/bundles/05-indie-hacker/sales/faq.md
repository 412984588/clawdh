# FAQ — Indie Hacker Bundle

**Is the Agency Blueprint for freelancers or agencies?**
Both. The templates scale from solo consultant to small agency. Most indie hackers use it to systematize client acquisition and delivery.

**What project types does Claude Code Project Kits cover?**
Web apps, APIs, CLI tools, SaaS products, and data pipelines. Each kit includes architecture decisions, file structure, and Claude Code workflow configuration.

**Is the Conference Prep Kit just for speakers?**
No. About half the content is for attendees: session selection, networking, note-taking, and follow-up. The speaker content is for those who want to submit proposals.
