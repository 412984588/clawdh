# FAQ — Knowledge Worker Bundle

**Do I need to already use Obsidian?**
No. The vault works as an introduction to Obsidian — structure and plugins are pre-configured. But if you already use Obsidian, you can merge the templates into your existing vault.

**Which Obsidian plugins are required?**
The vault is configured for: Copilot, Text Generator, Smart Connections, Templater, and Dataview. All are free.

**Are the Claude prompts only for Claude Code?**
Most prompts work in Claude.ai chat too. The skills (Claude Skills Vault) are Claude Code-specific.

**Can I use the skills without Claude Code?**
The skills require Claude Code CLI. The prompts work everywhere.
