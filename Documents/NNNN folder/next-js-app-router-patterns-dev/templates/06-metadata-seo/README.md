# 06 — Metadata & SEO

Static and dynamic `metadata`, Open Graph, OG image generation, structured data, `robots.txt`, and `sitemap.xml`.

## Patterns

- Static `metadata` export — title templates, OG, Twitter Card, canonical URLs
- Dynamic `generateMetadata()` — async metadata from route params / database
- `viewport` export — theme color, responsive viewport (Next.js 15)
- OG image generation — `ImageResponse` in `/api/og` route
- JSON-LD structured data — Article, Product, Organization schemas
- `robots.ts` — crawl rules and sitemap pointer
- `sitemap.ts` — dynamic XML sitemap generation
