# Changelog

## v1.0.0 — 2026-03-21

### Added
- 10 OpenAPI 3.1 spec templates
- 01-rest-crud: Standard CRUD with pagination and error schemas
- 02-auth-api: JWT auth, refresh tokens, forgot/reset password
- 03-ecommerce-api: Products, cart, orders, checkout
- 04-file-upload-api: Multipart upload and presigned URLs
- 05-payment-api: Stripe-compatible payments and subscriptions
- 06-social-api: Posts, likes, comments, follows, feed
- 07-search-api: Full-text search, autocomplete, facets
- 08-webhook-api: Webhook management, deliveries, test events
- 09-admin-api: User management, feature flags, audit logs
- 10-multi-tenant-api: Organizations, members, roles, invitations
