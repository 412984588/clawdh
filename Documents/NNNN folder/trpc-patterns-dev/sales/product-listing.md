# tRPC v11 Patterns — 10 Production-Ready Templates

Stop copy-pasting tRPC boilerplate. This pack gives you 10 battle-tested patterns covering everything from basic routers to real-time subscriptions and Next.js App Router integration.

## What You Get

**Starter ($19) — Templates 01–05**
- Router Basics: initTRPC, nested routers, input/output schemas
- Context & Middleware: JWT auth, role-based procedures, rate limiting
- Error Handling: TRPCError, custom error classes, errorFormatter
- Zod Input: transforms, refinements, discriminated unions
- Subscriptions: WebSocket observable, real-time chat, live metrics

**Pro ($39) — All 10 Templates**
Everything in Starter, plus:
- File Upload: presigned S3 URLs, metadata validation, batch uploads
- Batch Requests: httpBatchLink, splitLink, deduplication
- Testing tRPC: createCallerFactory, mock context, role-based test fixtures
- Next.js Integration: App Router, server components, React Query hooks
- Infinite Queries: cursor pagination, useInfiniteQuery, bidirectional loading

## Who This Is For

TypeScript developers building full-stack apps with tRPC v11, Next.js 14/15, or any Node.js backend.

## Requirements

- Node.js 18+, TypeScript 5+, @trpc/server v11, zod v3
