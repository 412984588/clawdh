# Svelte + SvelteKit Cursor Rules

## 适用场景
SvelteKit 2 + Svelte 5 应用，使用 TypeScript + Tailwind CSS。

## 核心规则摘要
- Svelte 5 Runes 语法（$state/$derived/$effect/$props）
- SvelteKit 文件约定（+page.svelte / +page.server.ts / +layout.svelte）
- Form Actions + enhance 渐进增强
- hooks.server.ts 实现认证
- load 函数服务端数据获取

## 使用方法
将 `.cursorrules` 文件复制到你的 SvelteKit 项目根目录。
