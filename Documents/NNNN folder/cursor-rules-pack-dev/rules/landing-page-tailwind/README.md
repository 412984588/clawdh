# Landing Page Tailwind Cursor Rules

## 适用场景
营销落地页/产品页，使用 Next.js + Tailwind CSS + Framer Motion。

## 核心规则摘要
- Hero/Features/SocialProof/FAQ/CTA 标准区块结构
- SEO 元数据（OpenGraph + Twitter Card）
- LCP 优化（next/image priority）
- 无障碍访问（语义 HTML + ARIA）
- 转化率优化（CTA 文案、社会证明）

## 使用方法
将 `.cursorrules` 文件复制到你的落地页项目根目录。
