# Cursor Rules Pack

**10 battle-tested `.cursorrules` + `.cursor/rules` files. Drop in, get smarter AI suggestions immediately.**

## The Problem

Cursor AI gives generic suggestions without context about your stack. Add the right rules and it starts
suggesting idiomatic code, catching anti-patterns, and following your project's conventions.

## Two Formats Per Template

- `.cursorrules` — legacy format (Cursor < 0.43)
- `.cursor/rules/main.mdc` — new MDC format (Cursor 0.43+)
