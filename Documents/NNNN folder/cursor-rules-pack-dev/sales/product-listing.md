# Cursor Rules Pack

**10 `.cursorrules` files for real stacks. Better AI suggestions, fewer corrections.**

## The Problem

Cursor suggests generic code. Without rules, it doesn't know your stack, conventions, or anti-patterns. You spend time correcting suggestions that should have been right the first time.

## The Solution

10 production-grade Cursor rule sets. Each one tells Cursor exactly how to write code for your stack: which patterns to follow, what to avoid, how to structure files.

## Two Formats Per Template

Every template includes both:
- `.cursorrules` — legacy format (Cursor < 0.43)
- `.cursor/rules/main.mdc` — new MDC format (Cursor 0.43+)
