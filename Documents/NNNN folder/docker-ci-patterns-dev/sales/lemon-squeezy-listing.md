# Lemon Squeezy Listing — Docker & CI Patterns for TypeScript

## Product Name
Docker & CI Patterns for TypeScript

## Short Description
10 production-ready Docker and CI/CD templates for TypeScript/Node.js projects. Copy-paste into your next deploy.

## Long Description
Tired of debugging the same Dockerfile issues on every project? This pack gives you 10 battle-tested templates that cover the full deployment lifecycle — from writing your first multi-stage Dockerfile to scaling with Kubernetes.

### Templates included:

| Template | What it solves |
|----------|---------------|
| 01 Dockerfile basics | .dockerignore, build args, programmatic builder |
| 02 Multi-stage builds | Shrink prod images: Node, Next.js, Prisma, FastAPI |
| 03 Docker Compose | Dev, staging, and prod compose stacks |
| 04 GitHub Actions CI | Tests with real PostgreSQL + Redis, coverage upload |
| 05 GitHub Actions Deploy | ECS, Railway, Fly.io, Render workflows |
| 06 Dockerfile optimization | BuildKit caching, distroless images |
| 07 Secrets management | Docker secrets, .env validation, OIDC |
| 08 Health monitoring | /health, Prometheus, Grafana |
| 09 Multi-service stack | Full microservice stack with message queue |
| 10 Kubernetes basics | Deployment, Service, Ingress, HPA |

- **Pro** ($39) — All 10 templates

### Tech stack:
TypeScript · Node.js · Docker · GitHub Actions · Kubernetes
