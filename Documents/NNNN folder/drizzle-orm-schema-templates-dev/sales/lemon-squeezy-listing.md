# Drizzle ORM Schema Templates Pack

10 production-ready Drizzle ORM database schemas. TypeScript. PostgreSQL.

## Two Tiers

**Starter — $19**
- Users & Auth (sessions, OAuth, email verification)
- Blog / CMS (posts, categories, tags, comments)
- SaaS Billing (plans, subscriptions, invoices, metered usage)
- E-Commerce (products, variants, orders, line items)
- Multi-Tenancy (orgs, memberships, invitations, API keys)

**Pro — $39**
All 5 Starter schemas plus:
- Analytics (page views, events, sessions, daily metrics)
- File Storage (files, folders, versions, S3/R2 sharing)
- Notifications (multi-channel, per-user prefs, push tokens)
- Audit Log (immutable trail, field-level diffs)
- Social Network (follows, posts, likes, hashtags)

## Requirements
- Drizzle ORM 0.30+
- drizzle-orm/pg-core
- PostgreSQL database
