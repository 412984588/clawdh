# Redis Patterns Dev Pack — itch.io Listing

**Title:** Redis Patterns Dev Pack

**Short description:** 10 TypeScript templates for production Redis patterns.

**Classification:** Tools / Developer Tools

**Description:**

A collection of TypeScript implementations covering the most important Redis patterns for backend development.

Each template is a standalone file you can copy into your project. No framework lock-in. Pure ioredis.

**Included templates:**
1. String operations — GET/SET, counters, JSON, bit operations
2. Hash maps — HSET/HGET/HGETALL, atomic increments, HSCAN
3. Lists & queues — FIFO, BLPOP, stack pattern, activity feed
4. Sets & sorted sets — set operations, leaderboard, time series log
5. Pub/Sub — basic channels, PSUBSCRIBE, event bus, presence
6. Caching strategy — cache-aside, write-through, stampede prevention (Pro)
7. Rate limiting — fixed/sliding window, token bucket with Lua (Pro)
8. Session store — create, slide, rotate, invalidate (Pro)
9. Distributed locks — Redlock, fencing tokens, leader election (Pro)
10. Streams & events — consumer groups, DLQ, event sourcing (Pro)
