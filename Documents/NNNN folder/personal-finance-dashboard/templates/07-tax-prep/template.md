## Document Checklist

Gather before filing:

**Income:**
- [ ] W-2(s) from all employers
- [ ] 1099-NEC / 1099-MISC (freelance)
- [ ] 1099-INT (bank interest)
- [ ] 1099-DIV (dividends)
- [ ] 1099-B (investment sales)
- [ ] 1099-K (payment platforms)
- [ ] 1099-R (retirement distributions)

**Deductions:**
- [ ] 1098 (mortgage interest)
- [ ] 1098-T (tuition)
- [ ] 1098-E (student loan interest)
- [ ] Charitable donation receipts
- [ ] Medical expense receipts
- [ ] Business expense records

**Other:**
- [ ] Last year's tax return (for reference)
- [ ] Social Security numbers (you + dependents)
- [ ] Bank account info (for direct deposit refund)
- [ ] Estimated tax payments made this year
