## Subscription Decision Framework

Before subscribing to anything new, answer these:

| Question | Answer |
|----------|--------|
| Will I use it more than 4x/month? | |
| Is there a free alternative? | |
| Does it save me more time/money than it costs? | |
| Can I share it with someone (family plan)? | |
| Does it have an annual discount? | |

**If 3+ answers are "No" → Don't subscribe.**
