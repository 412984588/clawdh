## Goal Planning Calculator

### How much do I need to save monthly?

```
Monthly contribution = (Target amount - Current savings) ÷ Months until deadline

Example: $10,000 goal, $2,000 saved, 12 months away
= ($10,000 - $2,000) ÷ 12 = $667/month
```

### How long will it take?

```
Months to goal = (Target - Current) ÷ Monthly contribution

Example: $5,000 goal, $500 saved, saving $300/month
= ($5,000 - $500) ÷ $300 = 15 months
```

## Savings Automation Checklist

The best savings system is one you don't have to think about:

- [ ] Set up automatic transfer to savings account (day after payday)
- [ ] Increase 401k/retirement contribution by 1% (do this annually)
- [ ] Set up separate sub-accounts for each goal (if your bank supports it)
- [ ] Enable round-up savings (Acorns, Qapital, or bank feature)
- [ ] Direct deposit split: send savings directly from paycheck
