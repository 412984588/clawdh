# Changelog

## [1.0.0] - 2024-01-01

### Added
- 10 production-ready React Email templates
- TypeScript support with typed props
- Two tiers: Starter (5 templates, $19) and Pro (10 templates, $39)
- Templates: welcome, verify-email, reset-password, magic-link, invoice
- Pro templates: subscription-started, subscription-cancelled, team-invitation, payment-failed, onboarding
- Dual ZIP builds (starter + pro) via build_bundle.py
