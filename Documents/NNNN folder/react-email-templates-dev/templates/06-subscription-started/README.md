# Subscription Started

Sent when a user starts a paid subscription.

## Props

| Prop | Type | Default |
|------|------|---------|
| `name` | string | `"User"` |
| `planName` | string | `"Pro"` |
| `amount` | number | `49` |
| `currency` | string | `"USD"` |
| `nextBillingDate` | string | `"2024-02-01"` |
| `dashboardUrl` | string | `"https://example.com/dashboard"` |
