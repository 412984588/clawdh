# Welcome Email

Sent immediately after a user creates an account.

## Props

| Prop | Type | Default |
|------|------|---------|
| `name` | string | `"there"` |
| `loginUrl` | string | `"https://example.com/login"` |

## Usage

```tsx
import WelcomeEmail from "./welcome";
<WelcomeEmail name="Alice" loginUrl="https://app.example.com/login" />
```
