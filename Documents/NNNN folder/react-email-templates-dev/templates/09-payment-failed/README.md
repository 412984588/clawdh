# Payment Failed

Sent when a subscription payment fails. Includes retry date and payment update CTA.

## Props

| Prop | Type | Default |
|------|------|---------|
| `name` | string | `"User"` |
| `amount` | number | `49` |
| `currency` | string | `"USD"` |
| `planName` | string | `"Pro"` |
| `retryDate` | string | `"2024-01-08"` |
| `updatePaymentUrl` | string | `"https://example.com/billing"` |
