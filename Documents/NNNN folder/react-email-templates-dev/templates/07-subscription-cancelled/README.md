# Subscription Cancelled

Sent when a user cancels their subscription. Includes access end date and reactivation CTA.

## Props

| Prop | Type | Default |
|------|------|---------|
| `name` | string | `"User"` |
| `planName` | string | `"Pro"` |
| `accessUntil` | string | `"2024-02-01"` |
| `reactivateUrl` | string | `"https://example.com/reactivate"` |
