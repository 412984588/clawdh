# Team Invitation

Sent when a user is invited to join a team or workspace.

## Props

| Prop | Type | Default |
|------|------|---------|
| `inviterName` | string | `"Alex"` |
| `teamName` | string | `"Acme Team"` |
| `inviteUrl` | string | `"https://example.com/invite/accept"` |
| `role` | string | `"Member"` |
| `expiresInDays` | number | `7` |
