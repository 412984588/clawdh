# React Email Templates Pack

10 production-ready transactional email templates built with React Email (TypeScript). Stop writing email HTML from scratch.

## Two Tiers

**Starter — $19**
- Welcome Email
- Verify Email (OTP + link)
- Reset Password
- Magic Link (dark mode)
- Invoice (multi-line-item)

**Pro — $39**
All 5 Starter templates plus:
- Subscription Started
- Subscription Cancelled
- Team Invitation
- Payment Failed (dunning)
- Onboarding (3-step)

## Features
- TypeScript with typed props
- Works with any Node.js email sender
- Compatible with React Email's preview server
- Mobile-responsive
- One-time purchase, unlimited projects

## Requirements
- React 18+
- `@react-email/components`
