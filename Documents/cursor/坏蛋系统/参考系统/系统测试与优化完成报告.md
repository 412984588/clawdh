## 🎯 性能优化策略

### 1. 阈值优化策略
- **原阈值过高**: 0.3导致大量漏检
- **新阈值平衡**: 0.15在召回率和精确率间找到平衡
- **分层检测**: 不同类型采用不同评分权重

### 2. 指标扩展策略
- **直接指标**: Stripe JS/API/Connect模式
- **间接指标**: 业务模式、电商特征
- **上下文指标**: 技术栈、网站类型
- **隐藏指标**: data属性、meta标签

### 3. 评分优化策略
```python
# 综合评分权重分配
overall_score = (
    direct_stripe_score * 0.4 +     # 直接证据权重最高
    contextual_score * 0.4 +        # 上下文证据次之
    technical_score * 0.2           # 技术证据权重较低
)
```
