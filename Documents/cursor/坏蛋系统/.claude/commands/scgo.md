# SCGO Slash Command

## 🎯 描述
Supreme Commander Go - SCGO超级指挥官快捷命令，使用斜杠命令格式发现并验证指定数量的美国支付平台。

## 📋 使用方法
```bash
/scgo [数量]
```

## 🚀 命令示例
```bash
/scgo 10     # 发现10个平台
/scgo 100    # 发现100个平台
/scgo 1000   # 发现1000个平台
/scgo        # 使用默认数量(20个)
```

## 🎖️ SCGO承诺
无论输入多大数字，哪怕是一万个平台，SCGO都会带领13个精英agent团队：
- 智能批次处理
- 动态策略优化
- 坚决完成任务
- 自动生成报告

## ⚡ 核心特性
- **批量处理**: 自动分解大任务为小批次
- **智能策略**: 根据任务特征选择最优策略
- **实时监控**: 显示执行进度和结果
- **ACE学习**: 自动总结经验并持续改进
- **承诺必达**: 无论数字多大，坚决完成

## 👥 执行团队
- **Supreme Commander (SC)**: 总指挥
- **13个精英agent**: 覆盖突破、分析、验证、学习等全流程

## 📊 支持命令
- `/scgo [数量]` - 发现并验证平台
- `/help` - 显示帮助信息
- `/status` - 显示系统状态
- `/team` - 显示agent团队状态
